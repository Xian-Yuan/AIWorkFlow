# UE 5.7 新特性

## 概述

UE 5.7 引入了多项重要更新，对 Lyra 开发产生影响。

## Lyra 5.7 相关更新

### Gameplay Framework Component Manager

- `UGameFrameworkComponentManager` 功能增强
- Initialization State 系统改进
- 更好的跨 Actor 依赖处理

### Common User Plugin

- `CommonUser` 插件更新
- 改进的在线服务集成

### Animation

- Lyra 动画系统更新
- 更好的 MetaHuman 兼容性

### Input

- `ULyraInputModifiers` 增强
- Enhanced Input 系统改进
- 更好的 Gamepad 支持

## 重要 API 变更 (追踪官方升级指南)
