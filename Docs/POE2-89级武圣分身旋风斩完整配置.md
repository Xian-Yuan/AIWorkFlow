## 三、装备配置（逐条列出）

### 武器1 — Quarterstaff（长棍）| 预算2-3D

**买装搜索条件**：
- 底子：Sinister Quarterstaff / Skullcrusher / Razor / Striking / Aegis / Lunar / Wyrm
- **物理DPS 800+**（最重要！）
- 暴击率 7%+
- +近战技能等级
- 攻击速度
- 暴击伤害加成
- **不要买 Dreaming Quarterstaff**（0%暴击率）

**省钱替代**：Runeforged Duality（符炼二元）选基础伤害版，900+物理DPS

### 武器2 — Talisman（护符）| 预算0.3-0.5D

- 推荐：Amor Mandragora（增加Rend技能持续时间）
- 任意Talisman也行，不需要花大钱

### 头盔 | 预算1-1.5D

**买装搜索条件**：
- 前缀：能量盾（尽量高）+ 闪避值（尽量高）+ 百分比闪避盾
- 后缀：全元素抗性 + 稀有度（Rarity）
- 额外：闪电竞值（配合石拳转化暴击率）

**关键操作**：
1. **先涂油再瓦！** 瓦了不能涂油
2. 涂油：**坚定偏转**（免疫流血）
3. 最终用 Raven-Touched Shard（渡鸦碎片）注入额外天赋
4. 用 Potent Liquid Melancholy + Ferocity + Contempt 激活 **Zarokh's Gift** → 多一个珠宝孔

### 胸甲 | 预算1-1.5D

**买装搜索条件**：
- 前缀：能量盾（尽量高）+ 闪避值 + 百分比闪避盾
- 后缀：全元素抗性 + 稀有度
- 特殊：免疫诅咒 + 能量盾充能速率

**目标数值**：1500闪避 + 500盾 或 900+盾

### 手套 — Lochtonial Caress（洛克顿之握）| 预算0.5-1D

**这是BD灵魂装备，必须买！**

**操作步骤**：
1. 买 Lochtonial Caress
2. 去 Verisium Anvil（威瑞希姆铁砧）符炼 → Stonefist版
3. 符炼会重roll数值，**只看暴击球词缀**
4. "5-10%几率在命中时为附近友方提供暴击球" → **必须≥8%**
5. 低于8%去铁砧重新符炼

### 鞋子 | 预算0.5-1D

**买装搜索条件**：
- 前缀：闪避值 + 能量盾 + 移动速度
- 后缀：**必须有一条"获得等同于X%闪避值的偏斜值"** + 元素抗性
- 额外：稀有度

### 腰带 | 预算0.5D

**买装搜索条件**（黄装腰带）：
- 前缀：闪避值 + 技能速度
- 后缀：全元素抗性 + 暴击率/暴击伤害
- 护符栏位越多越好

**护符选择**：
- **Rite of Passage（仪式通道）**：杀稀有怪给20秒强力buff
- **Aspect of the Cat（猫之面相）**：移速+暴击+技能速度（最强护符）
- 替代：狼/鹿/猫科护符（加速）

### 项链 | 预算1-1.5D

**买装搜索条件**：
- 前缀：**+3近战技能等级**（最重要！）+ 能量盾
- 后缀：暴击率 + 暴击伤害 + 全元素抗性
- 额外：稀有度 + 精魂（Spirit）
- 底子：Gold Amulet（黄金护符）最佳

**涂油**：
- Well of Power（力量之井）→ 解决魔力续航
- 或 剧毒耐受 → 免疫中毒（国服推荐）

### 戒指 x2 | 预算0.5-1D

**买装搜索条件**：
- 前缀：**攻击附加XX到XX伤害**（必须！）+ 稀有度
- 后缀：元素抗性 + 暴击率/暴击伤害
- 底子：Iron Ring（铁戒指，自带攻击附加伤害底子）

## 六、天赋树要点

### 核心天赋
- **Chaos Inoculation（混沌免疫）**：1血CI，免疫混沌伤害
- **Ghost Dance（幽灵之舞）**：回盾核心
- **Hollow Palm Technique（空掌术）**：开荒期用，后期可洗掉
- **Evasion（闪避）**：所有闪避相关节点都拿
- **Energy Shield（能量盾）**：所有盾相关节点都拿
- **Critical Strike（暴击）**：暴击率+暴击伤害
- **Area of Effect（范围）**：增大回旋击范围

### 武器套装天赋
- **武器套装2**：Quarterstaff技能（Hollow Form, Killing Palm等）
- **武器套装1**：持续时间节点（延长Rend/Pounce的buff/debuff）

### 属性节点
- 按装备需求补力量/敏捷/智慧
- 通常前期2点力量（红辅助宝石需求），然后平衡敏捷和智慧

## 九、操作手法

1. **清图**：按住 Hollow Form → 分身自动回旋击清屏
2. **跳怪**：Killing Palm 跳进下一堆怪

## 十一、常见问题

**Q: 89级Hollow Form有几个孔？**
A: 65级5孔，90级6孔。89级应该已有6孔（如果还没到90就5孔，去掉Rakiata's Flow）

**Q: 什么时候转CI？**
A: 能量盾达到3000+后转。转CI后把天赋树切到Endgame CI版

**Q: 洛克顿之握暴击球词缀低于8%怎么办？**
A: 去Verisium Anvil重新符炼，每次重roll，直到>=8%

**Q: 魔血(Mageblood)必须吗？**
A: 不必须！10D预算买不起，用黄装腰带+护符完全够用

**Q: 分身吃自己的友方增伤吗？**
A: 不吃！这是机制限制，分身吃不到自己的友方增伤

**Q: Whirling Assault宝石怎么升？**
A: 优先升等级，20%品质+1等级的瓦版最佳

**Q: Rigwald's Ferocity什么时候换？**
A: 当你有足够元素附加伤害来源（Heart of the Well+符文+Rend buff）时，约一半伤害变成非物理，这时换Rigwald比Heavy Swing强
