# 动画系统

## 概念

Lyra 动画系统基于 UE5 的 **IK Rig**、**Control Rig** 和 **动画蓝图**，支持模块化角色部件和 MetaHuman 兼容骨架。

## 关键特性

- 与 MetaHuman 共享核心骨架层次结构
- 使用 **Animation Sharing** 插件优化多角色动画
- **Modular Character Parts** — 可动态组合的角色部件（身体、头部等）
- 动画蓝图分层结构
