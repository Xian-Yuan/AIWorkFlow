## 一、按"实惠 + 不卡 + 效果好"排序的 Top 3

### 🏆 Top 1：fal.ai Seedance 2.0 Mini 720p（**今天就能用**）
- **价格**：¥5.6/条（720p / 5s）
- **质量**：Seedance 2.0 效果（接近官方 SOTA）
- **速度**：3-5 分钟出片，不排队（高峰期偶尔慢）
- **接入**：注册 fal.ai → GitHub 登录 → 创建 FAL_KEY → 装 `fal-client` SDK → 跑 Python 脚本（详见 `02-Seedance-2-API-Deep-Dive.md` 附录 B）
- **缺点**：需海外信用卡（Stripe）；国内卡失败率高
- **适用**：个人体验 / 短期项目 / 不想走企业认证

### 🥈 Top 2：火山方舟 Seedance 2.0 Lite（**国内最便宜，但需企业认证**）
- **价格**：推断 ¥0.6–¥0.9/条（720p / 5s，官方未公开）
- **质量**：Seedance 2.0 官方版本
- **速度**：高峰期排队 30s-5min，无明显限速
- **接入**：注册火山引擎 → **企业认证**（营业执照 + 对公账户打款 ¥1）→ 创建 API Key → 充值 ¥100 起
- **缺点**：必须企业认证（个人走不通，可淘宝找代理 ¥300-500）
- **适用**：商业项目 / 长期批量生产 / 有营业执照

### 🥉 Top 3：可灵 AI（快手）官方订阅（**国内最稳，套餐灵活**）
- **价格**：¥66/月起（基础包），付费档单条约 ¥0.5–¥1.5
- **质量**：Kling 2.6 Pro 在 Elo T2V 排行榜前列（仅次于 Seedance 2.0）
- **速度**：高峰期偶尔排队，但国内平台最快
- **接入**：下载可灵 AI App 或 Web → 订阅即可，无需 API
- **缺点**：无开放 API（仅 Web/客户端使用）
- **适用**：纯体验 / 不需要程序化调用 / 月量 < 200 条

## 三、Seedance 2.0 API 直连 vs 平台订阅 — 三条决策路径

### 路径 A：直接接 fal.ai（最省心，今天跑通）
```
注册 fal.ai（GitHub 登录）
  → 创建 API Key + 绑信用卡（Stripe）
  → pip install fal-client
  → 写 10 行 Python（详见 02-Seedance-2-API-Deep-Dive.md §3.5）
  → 跑通第一条视频 ¥2.6（Mini 480p）→ ¥5.6（Mini 720p）
```
- **优点**：今天就能跑、不卡、效果最好、海外支付稳定
- **缺点**：每条 ¥5.6-11.1，比国内贵 5-10 倍
- **适合**：个人体验 / 短期项目 / 不介意价格

### 路径 B：火山方舟 + 企业认证（国内最便宜，长期稳定）
```
注册火山引擎账号（个人可注册）
  → 申请企业认证（淘宝代理 ¥300-500 或自己办个体户 ¥200-300）
  → 1-3 工作日通过
  → 在控制台勾选 Seedance 2.0 Lite
  → 充值 ¥100 起 → 创建 AK/SK
  → 调 OpenAI 兼容 API（详见 02-Seedance-2-API-Deep-Dive.md §4.4）
```
- **优点**：推断 ¥0.6-0.9/条，长期最便宜；中文支持；合规
- **缺点**：企业认证门槛（个人走不通）；价格不透明
- **适合**：商业项目 / 月量 1000+ 条 / 长期使用

### 路径 C：自部署 Wan2.1 14B（极限省钱，但要 GPU）
```
硬件：RTX 4090（24GB ¥0.99/h 整机）或 H100（80GB）
  → git clone https://github.com/Wan-Video/Wan2.1
  → pip install -r requirements.txt
  → 下载 Wan2.1-T2V-14B 模型（80GB 磁盘）
  → 跑 Python 脚本 / 起 FastAPI 服务
  → 单条 ¥0.4（边际电费成本）
```
- **优点**：零边际成本（除电费）；不受第三方涨价影响
- **缺点**：需 24GB+ 显卡；首次部署 1-2 周；效果 80% 接近 Seedance 2.0
- **适合**：月量 > 4000 条 / 有 GPU 闲置 / 想长期控制成本

## 五、常见误区澄清

### ❌ "Sora 2 是不是最强？"
**答**：Sora 2 效果顶级（Elo 1238），但 **OpenAI 已于 2026-04-26 停服 Sora 1 网页/APP**（API 2026-09-24 停服）。Sora 2 还在，但价格高、限速严、排队严重。**目前不推荐作为主力**。

### ❌ "Replicate 上能跑 Seedance 2.0 吗？"
**答**：**不能**。Replicate 只有 Seedance 1.0 Pro / Lite，没有 2.0。**字节只授权 fal.ai 和自家火山方舟**。

### ❌ "阿里云百炼能跑 Seedance 2.0 吗？"
**答**：**不能**。阿里云百炼只有通义万相 Wan 系列，没有 Seedance。

### ❌ "自部署 Seedance 2.0 行不行？"
**答**：**不行**。Seedance **没有开源**。最接近的开源替代是 Wan2.1 14B（效果 80% 接近）。

### ❌ "Hailuo 海螺 = Seedance？"
**答**：**不是**。海螺 AI 是 MiniMax 公司产品，不是字节。模型不同。
