# 直链生成与支付税务

## 概述

插件打包后需要生成加密直链上传给 Epic 审核团队，同时需要了解支付模型、税务设置和审核流程。

---

## 虚幻商城直链生成

### Google Drive 直链

商城审核不支持直接上传文件，需要提供商直链（Direct Download Link）。Google Drive 是最常用的方式。

**生成步骤：**

1. 将打包文件（`.uplugin` 或 `.zip`）上传到 Google Drive
2. 获取分享链接（格式类似：`https://drive.google.com/file/d/FILE_ID/view?usp=sharing`）
3. 替换链接格式为直链：
   ```
   原始：https://drive.google.com/file/d/FILE_ID/view?usp=sharing
   直链：https://drive.google.com/uc?export=download&id=FILE_ID
   ```

### 链接安全要求

- 直链**不能公开暴露**，仅提供给 Epic 审核团队
- 不要将链接发布到公开论坛、社交媒体或文档中
- 审核完成后，如无问题可以关闭分享

### 支持的打包格式

- 仅接受 `.uplugin` 或 `.zip` 格式
- 确保打包文件完整性

---

## 版本支持配置

| 规则 | 说明 |
|------|------|
| 单链接单版本 | 每个直链对应**一个引擎版本** |
| 最新 3 版支持 | 必须支持最新的 3 个引擎大版本 |
| 不可回溯 | 超过 3 个版本的旧版不再支持，不可回溯补支持 |

- 例如：当最新引擎为 UE 5.5 时，需同时提供 UE 5.3、5.4、5.5 三个版本的链接
- 每个版本单独打包、单独生成直链

## 支付模型

### Epic 抽成

| 项目 | 数值 |
|------|------|
| **Epic 抽成比例** | 12% |
| **开发者所得** | 88% |

### 付款条件

| 条件 | 说明 |
|------|------|
| **月付门槛** | 累计收入超过 $100 时按月支付 |
| **年付** | 累计收入低于 $100 时按年结算一次 |
| **结算滞后** | 30 天结算延迟（例如 1 月的销售，3 月到账） |

### 银行推荐

- 推荐使用**招商银行一卡通（储蓄卡）**
- 支持 Swift Code 国际汇款
- Hyperwallet 可直接汇款到该账户

---

## 审核流程

| 阶段 | 说明 |
|------|------|
| **提交审核** | 在 Apex Seller Dashboard 提交商品信息和直链 |
| **首次响应** | 一般 2-3 个工作日收到回复（最慢的一次） |
| **反复修改** | 根据审核反馈修改后重新提交 |
| **审核通过** | 商品上架到虚幻商城 |

### 首次审核建议

- 首次审核通常是最慢的，后续更新会快很多
- 建议在提交前对照指南文档逐条自查，减少往返修改次数
- 特别注意：版权声明、第三方库归属、图片/描述语言

### 审核拒绝的常见原因

1. 商品截图或描述包含非英文内容
2. 文件缺少版权声明头
3. 商品标题包含商标名（UE、Apex、国家名称等）
4. 使用了未声明的第三方库
5. 代码质量不满足 IWYU 标准
6. 插件无法在最新引擎版本上正常工作
