# 远程攻击与弹道系统

## 架构设计

项目没有独立的 Bullet 或 Projectile C++ 类。远程攻击的能力通过 `AXGRPGHitBox_ApplyGameEffect` 结合 `UProjectileMovementComponent` 和 `EXGRPGHitCollisionType` 枚举实现。

这样的设计将碰撞检测、弹道移动和伤害应用整合到一个 Actor 中，避免创建多个独立的弹道类。

## 远程攻击能力

### 能力创建

远程攻击（魔法/子弹）作为独立的 GameplayAbility 存在：

- 通过 InputTag 绑定到按键
- 激活时设置 Targeting 参数（方向、目标位置）
- 生成 `AXGRPGHitBox_ApplyGameEffect` Actor
- 配置碰撞类型为 `LINE`（直线）或 `TRACK_LINE`（追踪）

### 子弹生成流程

```
远程攻击 GA 激活
    → 获取角色前向向量/目标方向
    → SpawnActor<AXGRPGHitBox_ApplyGameEffect>
        → PreInitCollision() 设置碰撞类型
        → 设置 ProjectileMovementComponent 参数（速度、方向）
        → 设置碰撞盒子大小
        → OnBeginOverlap → 检测到目标
            → 判重（IsExist）
            → 应用 GE（EffectClass）→ 触发伤害
```

## 弹道类型

| 碰撞类型 | 弹道行为 | ProjectileMovement 配置 |
|---------|---------|----------------------|
| `LINE` | 直线飞行，不追踪 | 初始速度 + 无 Homing |
| `TRACK_LINE` | 追踪目标 | 开启 Homing + 加速度 |
| `RANGE_LINE` | 投掷物抛物线 | 初始速度 + 重力 |
| `DIRECT_LINE` | 穿透性直线 | 高速度 + 穿透 |

## 魔法值消耗

远程攻击（魔法）消耗魔法值，通过 GameplayEffect 的 Modifier 实现：

1. 在 GE 中配置 Modifier 为 Mana 属性
2. Modifier Op 设为 `Additive`（负数表示扣除）
3. Magnitude 配置消耗量（可用 SetByCaller 实现动态消耗）
4. 将 GE 绑定到远程攻击能力的 `Cost GameplayEffect` 槽位

## 碰撞类型预初始化

```cpp
void AXGRPGHitBox_ApplyGameEffect::PreInitCollision(AActor* Owner)
{
    switch (HitCollisionType)
    {
        case EXGRPGHitCollisionType::HITCOLLISIONTYPE_LINE:
            // 设置直线弹道参数
            ProjectileMovement->Velocity = Owner->GetActorForwardVector() * Speed;
            ProjectileMovement->ProjectileGravityScale = 0.0f;
            break;
        case EXGRPGHitCollisionType::HITCOLLISIONTYPE_TRACK_LINE:
            // 设置追踪弹道参数
            ProjectileMovement->bIsHomingProjectile = true;
            ProjectileMovement->HomingAccelerationMagnitude = 3000.0f;
            break;
        // ...
    }
}
```

## 注意事项

- 远程攻击的子弹 Actor 仅存在于服务端，客户端通过复制看到其位置
- `ProjectileMovementComponent` 的网络同步模式需设为 `Interpolated` 以平滑客户端位置
- 子弹的碰撞检测应在服务端执行，客户端只做可视化表现
- 魔法消耗 GE 在能力激活时应用，如果魔法值不足则能力不应被激活（通过 `CheckCost` 检查）
- 远程攻击的追踪逻辑通过 `HomingTargetComponent` 实现，需在生成子弹时设置目标
