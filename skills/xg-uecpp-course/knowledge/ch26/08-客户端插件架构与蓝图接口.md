# 客户端插件架构与蓝图接口

## 概述

XGSampleClient 是一个 UE 插件，安装在游戏项目 `001_XGSampleDemo` 中。它采用 **BPLibrary + Subsystem + Delegate 回调** 的架构模式，向蓝图暴露登录/注册接口，底层通过 `FHttpModule` 发送 HTTP 请求。

## 架构：三层分离

```
蓝图节点调用
    │
    ▼
┌─────────────────────────────────────────┐
│ BPLibrary（第一层：蓝图入口）             │
│ - UCLASS BlueprintFunctionLibrary        │
│ - 静态方法，蓝图可直接搜索调用             │
│ - 创建请求数据 + 调用 Subsystem           │
└─────────────────┬───────────────────────┘
                  │
                  ▼
┌─────────────────────────────────────────┐
│ Subsystem（第二层：请求管理）             │
│ - UGameInstanceSubsystem                │
│ - 管理请求 ID + Delegate 映射            │
│ - 发送 HTTP 请求 + 接收响应              │
│ - 响应到达时查找 Delegate 并回调          │
└─────────────────┬───────────────────────┘
                  │
                  ▼
┌─────────────────────────────────────────┐
│ Type + Util（第三层：数据层）             │
│ - 共享 USTRUCT 类型定义                  │
│ - Token 加密工具                         │
│ - JSON 序列化/反序列化                   │
└─────────────────────────────────────────┘
```

## 第一层：BPLibrary（蓝图入口）

路径：[XGSampleClientBPLibrary.h](../code/001_XGSampleDemo/Plugins/XGSampleClient/Source/XGSampleClient/Public/XGSampleClientBPLibrary.h)，[XGSampleClientBPLibrary.cpp](../code/001_XGSampleDemo/Plugins/XGSampleClient/Source/XGSampleClient/Private/XGSampleClientBPLibrary.cpp)

```cpp
UCLASS()
class UXGSampleClientBPLibrary : public UBlueprintFunctionLibrary
{
    GENERATED_BODY()

    UFUNCTION(BlueprintCallable, Category = "XGSampleClient")
    static void Login(const FString& InUserName, const FString& InPassword,
        const FOnLoginResponse& InOnLoginResponse);

    UFUNCTION(BlueprintCallable, Category = "XGSampleClient")
    static void Register(const FString& InUserName, const FString& InPassword,
        const FString& InPhone, const FString& InMail,
        const FOnRegisterResponse& InOnRegisterResponse);
};
```

BPLibrary 的职责：
- 接收蓝图的参数输入
- 组装业务请求数据（`FXGSampleServerLoginRequestData` / `FXGSampleServerRegisterRequestData`）
- 将业务数据序列化为 JSON 字符串
- 调用 Subsystem 发送请求
- 将蓝图传入的 Delegate 转发给 Subsystem

实现示例（登录）：

```cpp
void UXGSampleClientBPLibrary::Login(const FString& InUserName,
    const FString& InPassword, const FOnLoginResponse& InOnLoginResponse)
{
    // 1. 组装业务请求
    FXGSampleServerLoginRequestData RequestData;
    RequestData.UserName = InUserName;
    RequestData.Password = InPassword;

    FString BusinessJsonStr;
    FJsonObjectConverter::UStructToJsonObjectString(RequestData, BusinessJsonStr);

    // 2. 包装到一级请求结构
    FXGSampleServerRequest OuterRequest;
    OuterRequest.Type = EXGSampleRequestType::LoginReq;
    OuterRequest.Data = BusinessJsonStr;

    // 3. 获取 Subsystem 发起请求
    UXGSampleClientSubsystem* Subsystem = GetSubsystem();
    Subsystem->SendHttpRequest(OuterRequest, InOnLoginResponse);
}
```

## 第二层：Subsystem（请求管理）

路径：[XGSampleClientSubsystem.h](../code/001_XGSampleDemo/Plugins/XGSampleClient/Source/XGSampleClient/Public/Subsystem/XGSampleClientSubsystem.h)，[XGSampleClientSubsystem.cpp](../code/001_XGSampleDemo/Plugins/XGSampleClient/Source/XGSampleClient/Private/Subsystem/XGSampleClientSubsystem.cpp)

### 声明

```cpp
UCLASS()
class UXGSampleClientSubsystem : public UGameInstanceSubsystem
{
    GENERATED_BODY()

    virtual void Initialize(FSubsystemCollectionBase& Collection) override;
    virtual void Deinitialize() override;

    void SendHttpRequest(const FXGSampleServerRequest& InRequestData,
        const FOnLoginResponse& InOnLoginResponse);
    void SendHttpRequest(const FXGSampleServerRequest& InRequestData,
        const FOnRegisterResponse& InOnRegisterResponse);

    void OnResponseReceived(FHttpRequestPtr InRequest,
        FHttpResponsePtr InResponse, bool bConnectedSuccessfully);

    int32 RequestIDCounter;
    TMap<int32, FOnLoginResponse> LoginDelegateMap;
    TMap<int32, FOnRegisterResponse> RegisterDelegateMap;
};
```

### HTTP 请求发送

```cpp
void UXGSampleClientSubsystem::SendHttpRequest(
    const FXGSampleServerRequest& InRequestData,
    const FOnLoginResponse& InOnLoginResponse)
{
    // 1. 生成唯一请求 ID
    int32 RequestID = RequestIDCounter++;

    // 2. 保存回调 Delegate
    LoginDelegateMap.Add(RequestID, InOnLoginResponse);

    // 3. 序列化请求 JSON
    FString RequestJsonStr;
    FJsonObjectConverter::UStructToJsonObjectString(InRequestData, RequestJsonStr);

    // 4. 创建 HTTP 请求
    FHttpRequestRef Request = FHttpModule::Get().CreateRequest();
    Request->SetURL(ServerURL);
    Request->SetVerb(TEXT("POST"));
    Request->SetHeader(TEXT("Content-Type"), TEXT("application/json; charset=utf-8"));

    // 5. 设置 Token Header
    FString CurrentToken = UXGSampleServerUtil::EncryptionToken(
        UXGSampleClientSettings::Get()->Token,
        UXGSampleClientSettings::Get()->MD5Num
    );
    Request->SetHeader(TEXT("XGLoginServerToken"), CurrentToken);

    // 6. 设置请求体
    Request->SetContentAsString(RequestJsonStr);

    // 7. 绑定响应回调 + 传递请求 ID
    Request->OnProcessRequestComplete().BindUObject(this,
        &UXGSampleClientSubsystem::OnResponseReceived, RequestID);

    // 8. 发送
    Request->ProcessRequest();
}
```

### 响应回调分发

```cpp
void UXGSampleClientSubsystem::OnResponseReceived(FHttpRequestPtr InRequest,
    FHttpResponsePtr InResponse, bool bConnectedSuccessfully, int32 InRequestID)
{
    if (!bConnectedSuccessfully || !InResponse.IsValid())
    {
        // 网络错误处理
        OnLoginResponse FailureResponse;
        FailureResponse.Result = EXGSampleServerResult::UnknownError;
        FailureResponse.Message = TEXT("Network Error");

        if (auto* Delegate = LoginDelegateMap.Find(InRequestID))
        {
            Delegate->ExecuteIfBound(FailureResponse);
            LoginDelegateMap.Remove(InRequestID);
        }
        return;
    }

    FString ResponseJsonStr = InResponse->GetContentAsString();
    FXGSampleServerLoginResponseData ResponseData;
    FJsonObjectConverter::JsonObjectStringToUStruct(ResponseJsonStr,
        &ResponseData, 0, 0);

    if (auto* Delegate = LoginDelegateMap.Find(InRequestID))
    {
        Delegate->ExecuteIfBound(ResponseData);
        LoginDelegateMap.Remove(InRequestID);
    }
}
```

Delegate Map 模式：
- **RequestIDCounter** 自增 ID，确保每个请求有唯一标识
- **Delegate Map** 存储请求 ID → Delegate 的映射
- **响应到达时**：查找对应 ID 的 Delegate → 执行 → 从 Map 中移除
- **自动清理**：Delegate 执行后立即从 Map 移除，防止内存泄漏

## 客户端配置

路径：[XGSampleClientSettings.h](../code/001_XGSampleDemo/Plugins/XGSampleClient/Source/XGSampleClient/Public/Config/XGSampleClientSettings.h)

```cpp
UCLASS(Config = Game, DefaultConfig)
class UXGSampleClientSettings : public UDeveloperSettings
{
    GENERATED_BODY()

    UPROPERTY(Config, EditAnywhere, Category = "Server")
    FString ServerURL = TEXT("http://127.0.0.1:11235/Login");

    UPROPERTY(Config, EditAnywhere, Category = "Security")
    FString Token = TEXT("XiaoGangNB");

    UPROPERTY(Config, EditAnywhere, Category = "Security")
    int32 MD5Num = 5;
};
```

使用 UE 的 `UDeveloperSettings` 体系，可以在 Project Settings 中可视化配置。配置字段包括：
- `ServerURL` — 服务器地址（IP + 端口 + 路径）
- `Token` — 与服务器一致的 Token 基准值
- `MD5Num` — 与服务器一致的 MD5 迭代次数

## 请求发送流程总结

```
蓝图调用 Login(UserName, Password, Delegate)
    │
    ▼
BPLibrary::Login()
    ├── 组装 FXGSampleServerLoginRequestData
    ├── 序列化为 JSON (FJsonObjectConverter)
    ├── 包装到 FXGSampleServerRequest (Type + Data)
    └── 调用 Subsystem->SendHttpRequest()
            │
            ▼
Subsystem::SendHttpRequest()
    ├── 生成 RequestID
    ├── 保存 Delegate → LoginDelegateMap
    ├── 序列化外层请求 JSON
    ├── 创建 FHttpRequest (FHttpModule)
    ├── 设置 URL/Verb/Header/Token
    ├── 绑定 OnResponseReceived(RequestID)
    └── ProcessRequest()
            │
            ▼
    [HTTP 请求发出，等待服务端响应]
            │
            ▼
Subsystem::OnResponseReceived()
    ├── 解析响应 JSON
    ├── 反序列化为 FXGSampleServerLoginResponseData
    ├── 查找 LoginDelegateMap[RequestID]
    ├── Delegate->ExecuteIfBound(ResponseData)
    └── 移除 LoginDelegateMap[RequestID]
```
