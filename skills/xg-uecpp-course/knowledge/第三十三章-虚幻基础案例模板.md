## 核心知识点

### 1. 增强输入系统（Enhanced Input）

本章完整演示了 Enhanced Input 的三种核心资产的使用：
- **Input Action**：定义输入类型（Boolean / 2D Axis）和触发行为（Triggered / Started / Completed）
- **Input Mapping Context (IMC)**：将 Input Action 映射到具体按键，支持 Modifier（Negate、SwapAxis）和优先级
- **本地玩家子系统**：`UEnhancedInputLocalPlayerSubsystem` 管理 IMC 的注册/移除
- **C++ 绑定**：SetupPlayerInputComponent → Cast 到 `UEnhancedInputComponent` → `BindAction()`
- 详见 [ch33/01-FP第一人称模板详解](ch33/01-FP第一人称模板详解.md)

### 2. FP（第一人称）模板

最完整的手写实现，涵盖全部六个 C++ 类：

| 类 | 继承 | 职责 |
|----|------|------|
| `FPC_SimpleGameMode` | `AGameModeBase` | 仅设置默认类，无业务逻辑 |
| `FPC_SimplePlayerController` | `APlayerController` | BeginPlay 注册 IMC |
| `FPC_SimpleCharacter` | `ACharacter` | Capsule→SpringArm→Camera→Mesh1P，输入绑定，Move/Look |
| `FPC_SimplePickupComponent` | `USphereComponent` | 重叠检测 → OnPickUp 委托 |
| `FPC_SimpleWeaponComponent` | `USkeletalMeshComponent` | 装备武器、注册武器 IMC、开火生成投掷物 |
| `FPC_SimpleProjectile` | `AActor` | SphereComponent + ProjectileMovementComponent，OnHit 销毁 |

- 详尽的 C++ 实现细节见 [ch33/01-FP第一人称模板详解](ch33/01-FP第一人称模板详解.md)

### 3. TP（第三人称）模板

- 共享与 FP 相同的 GameMode → PlayerController → Character 框架
- **差异**：CameraBoom + FollowCamera（第三人称跟拍）、IMC 注册在 Character::BeginPlay、全身骨骼网格体 + 动画状态机
- 本项目将作为后续多人网络章节的起点
- 详见 [ch33/02-TP第三人称模板概述](ch33/02-TP第三人称模板概述.md)

### 4. TD（俯视角）模板

- **核心交互**：鼠标点击移动（按住持续移动 / 单击寻路移动）
- **PlayerController**：`bShowMouseCursor = true`，LineTrace 检测光标下的 HitResult
- **角色设置**：`bOrientRotationToMovement = true`（移动跟随转向）
- **构建依赖**：需要 `AIModule`（SimpleMoveToLocation）+ `NiagaraSystem`（点击特效）
- 详见 [ch33/03-TD俯视角模板概述](ch33/03-TD俯视角模板概述.md)
