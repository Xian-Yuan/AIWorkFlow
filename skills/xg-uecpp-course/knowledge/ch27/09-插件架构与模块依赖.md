# 插件架构与模块依赖

## 两插件设计

本章功能拆分为两个插件，职责分离：

```
XGSamplePicture（摄像头 + PNG 生成）
    ↓ 依赖
XGSampleLink（科大讯飞 HTTP 通讯 + 加密）
    ├── XGSampleXFLink（讯飞模块：活体检测/STT/TTS）
    └── XGSampleBDLink（百度模块：其他章节功能）
```

## XGSampleXFLinkLibrary 加密库

位于 `Source/ThirdParty/XGSampleXFLinkLibrary/`，使用 ThirdParty 库类型：

### Build.cs 特点

代码见 [XGSampleXFLinkLibrary.Build.cs](file:///d:/UPS/GitLab/XGUnrealNote/Courses/UE-CPP-Basic-to-Advanced/code/001_XGSampleDemo/Plugins/XGSampleLink/Source/ThirdParty/XGSampleXFLinkLibrary/XGSampleXFLinkLibrary.Build.cs)：

```cpp
// 条件编译 OpenSSL
if (bPlatformSupportsSSL)
{
    PublicDefinitions.Add("WITH_SSL=1");
    AddEngineThirdPartyPrivateStaticDependencies(Target, "OpenSSL");
}
else
{
    PublicDefinitions.Add("WITH_SSL=0");
}
```

- 通过 `AddEngineThirdPartyPrivateStaticDependencies` 链接 UE 内置的 OpenSSL 静态库
- 跨平台支持：Win64/Mac/Linux/iOS/Android
- 宏标记：`SSL_PACKAGE=1` 用于统一启用标记，`WITH_SSL=1` 用于代码级条件编译

## 函数库

`XGSampleXFLinkBase` 类（[头文件](file:///d:/UPS/GitLab/XGUnrealNote/Courses/UE-CPP-Basic-to-Advanced/code/001_XGSampleDemo/Plugins/XGSampleLink/Source/ThirdParty/XGSampleXFLinkLibrary/Public/XGSampleXFLinkBase.h)）提供以下静态函数：

| 函数 | 用途 | 来源 |
|------|------|------|
| `HmacSha256()` | 原始 HMAC-SHA256 | OpenSSL `HMAC(EVP_sha256(), ...)` |
| `XGHMACSHA256()` | 讯飞签名 → Base64 | 本章 |
| `BDHMACSHA256()` | 百度签名 → Hex | 其他章节 |
| `AssembleAuthUrl()` | 完整 URL 签名 | 本章 |
| `URLEncode()` | URL 编码 | 本章 |
| `Base64UTF8StringToTCharString()` | Base64 → UTF8 → FString | 本章 |
| `GenerateBDHeaders()` | 百度鉴权 Header | 其他章节 |
| `GenerateBDAuthorizationToken()` | 百度 Token 生成 | 其他章节 |

### OpenSSL 包含方式

```cpp
THIRD_PARTY_INCLUDES_START
#include <openssl/hmac.h>
#include <openssl/sha.h>
#include <openssl/evp.h>
THIRD_PARTY_INCLUDES_END
```

使用 `THIRD_PARTY_INCLUDES_START/END` 宏包裹，禁用第三方头文件的警告，避免与 UE 自身的警告设置冲突。
