# 音频重采样与格式转换

## 需求背景

讯飞 STT 服务器要求音频格式为 **16KHz、16bit、单声道、PCM S16LE**（小端有符号 16bit）。

麦克风采集的原始音频通常是：
- 采样率：48000Hz 或 44100Hz
- 通道数：2（立体声）或 1（单声道）
- 格式：float 数组（值域 [-1.0, 1.0]）

因此需要经历两级转换：**重采样**（降采样到 16KHz + 混音到单声道）和 **位深转换**（float → int16 → uint8）。

## GetVoiceData：消费入口

`GetVoiceData` 是音频采集子系统暴露给消费线程的接口，从缓冲区取出数据进行重采样：

```cpp
bool UXGSampleAudioCaptureSubsystem::GetVoiceData(TArray<float>& OutVoiceData)
{
    FScopeLock Lock(&VoiceDataSection);

    if (AudioCaptureVoiceData.Num() <= 0)
        return false;

    // 根据源采样率选择处理方式
    if (VoiceSampleRate == 48000)
    {
        // 48KHz → 16KHz：按比例降采样
        // 48K stereo → 取左声道，每 3 取 1 → 16K mono
        float tempSample[1024];
        int32 TempIndex = 0;
        for (int32 i = 0; i < AudioCaptureVoiceData.Num() && TempIndex < 1024; i += VoiceChannelNum * 3)
        {
            tempSample[TempIndex++] = AudioCaptureVoiceData[i];
        }
        OutVoiceData.Append(tempSample, TempIndex);
    }
    else if (VoiceSampleRate == 44100)
    {
        // 44.1KHz → 16KHz：线性插值重采样
        // 44.1K stereo → 线性插值到 16K mono
        TArray<float> ResampleData;
        LinearResample(AudioCaptureVoiceData, VoiceChannelNum, VoiceSampleRate, 16000, ResampleData);
        OutVoiceData.Append(ResampleData);
    }

    AudioCaptureVoiceData.Empty(); // 清空已消费数据
    VoiceSampleRate = 0;
    VoiceChannelNum = 0;

    return OutVoiceData.Num() > 0;
}
```

## 48KHz → 16KHz（步进降采样）

对于 48KHz 立体声（2ch）输入：
- 3 倍降采样：48K / 16K = 3
- 取左声道：`AudioCaptureVoiceData[i]`（i 从 0 开始，每次跳过 `VoiceChannelNum * 3` 个元素）
- 输出数组上限 1024 个 float（即每帧约 64ms 数据 @16KHz）

```
原始：L0 R0 L1 R1 L2 R2 L3 R3 L4 R4 L5 R5 ...
取：   L0         L3         L6         ...
```

## 44.1KHz → 16KHz（线性插值重采样）

44.1KHz 到 16KHz 的比率不是整数（44.1 / 16 = 2.75625），无法用步进降采样，需要使用**线性插值算法**。

### LinearResample 实现

```cpp
void UXGSampleAudioCaptureSubsystem::LinearResample(
    const TArray<float>& InAudio, int32 InNumChannels,
    int32 InSampleRate, int32 OutSampleRate,
    TArray<float>& OutAudio)
{
    // 1. 先将立体声混音为单声道
    TArray<float> InputMono;
    if (InNumChannels == 2)
    {
        InputMono.SetNum(InAudio.Num() / 2);
        for (int32 i = 0; i < InputMono.Num(); i++)
            InputMono[i] = (InAudio[i * 2] + InAudio[i * 2 + 1]) * 0.5f; // 双声道平均
    }
    else
    {
        InputMono = InAudio;
    }

    // 2. 计算输出长度
    float ResampleScale = (float)OutSampleRate / (float)InSampleRate; // e.g. 16000/44100 ≈ 0.3628
    int32 OutputLength = FMath::CeilToInt(InputMono.Num() * ResampleScale);
    OutAudio.SetNum(OutputLength);

    // 3. 线性插值
    for (int32 OutIdx = 0; OutIdx < OutputLength; OutIdx++)
    {
        float SrcPos = (float)OutIdx / ResampleScale;          // 逆算源位置
        int32 SrcIdx = (int32)SrcPos;                          // 整数部分（左边界）
        float Fraction = SrcPos - SrcIdx;                      // 小数部分（插值权重）

        if (SrcIdx + 1 < InputMono.Num())
        {
            // 线性插值：out = left + (right - left) * fraction
            OutAudio[OutIdx] = InputMono[SrcIdx] + (InputMono[SrcIdx + 1] - InputMono[SrcIdx]) * Fraction;
        }
        else
        {
            OutAudio[OutIdx] = InputMono[SrcIdx];              // 边界处理：取最后一个值
        }
    }
}
```

插值原理：
```
源序列:  a0    a1    a2    a3    a4    ...
         ↓     ↓     ↓     ↓     ↓
         ↑-----↑-----↑-----↑-----↑
目标:    b0   b1   b2   b3   b4   ...
         ↑
       逆算 SrcPos = OutIdx / ResampleScale
       取 SrcIdx = floor(SrcPos), Fraction = SrcPos - SrcIdx
       b[OutIdx] = a[SrcIdx] + (a[SrcIdx+1] - a[SrcIdx]) * Fraction
```

## float → int16 → uint8 转换（SendVoiceData）

消费线程取出重采样后的 float 数据后，通过 `SendVoiceData` 转换为 WebSocket 可发送的二进制格式：

```cpp
void UXGSampleSTTSubsystem::SendVoiceData(const TArray<float>& InVoiceData)
{
    // 每帧固定 640 个 float（40ms @16KHz）
    // float 范围 [-1.0, 1.0] → int16 范围 [-32768, 32767]
    // int16 → 小端 uint8 字节

    TArray<uint8> BinaryData;
    BinaryData.Reserve(InVoiceData.Num() * 2); // 每个 float → 2 字节

    for (int32 i = 0; i < InVoiceData.Num(); i++)
    {
        // clamp 到 [-1.0, 1.0]
        float Clamped = FMath::Clamp(InVoiceData[i], -1.0f, 1.0f);
        // float → int16
        int16 Sample = static_cast<int16>(Clamped * 32767.0f);
        // int16 → 小端 uint8（2 字节）
        // Windows x64 本身就是小端，直接取低 8 位和高 8 位
        BinaryData.Add(static_cast<uint8>(Sample & 0xFF));          // LSB
        BinaryData.Add(static_cast<uint8>((Sample >> 8) & 0xFF));  // MSB
    }

    // 二进制发送
    Socket->Send(BinaryData.GetData(), BinaryData.Num(), true); // true = isBinary
}
```

转换公式：
| 阶段 | 公式 | 说明 |
|------|------|------|
| float → int16 | `sample_i16 = float * 32767` | 浮点值域映射到 int16 满量程 |
| int16 → LE bytes | `b[0] = sample & 0xFF` | 低字节 |
| | `b[1] = (sample >> 8) & 0xFF` | 高字节 |

## 数据帧大小

| 参数 | 值 |
|------|-----|
| 目标采样率 | 16000 Hz |
| 每帧样本数 | 640 个 float（约 40ms） |
| 每帧字节数 | 1280 字节（640 int16 × 2 字节） |
| 发送间隔 | 约 40ms（由消费线程 10ms 轮询 + 缓冲区积累决定） |
