# 麦克风音量计算

## RMS 算法

音量使用 RMS（Root Mean Square，均方根）算法计算，公式：

```
Volume = sqrt( Σ(sample[i]^2) / N )
```

- `sample[i]`：第 i 个样本值（float，[-1.0, 1.0]）
- `N`：样本总数
- 结果范围：[0.0, 1.0]，0 表示静音，1 表示满量程

## 实现

```cpp
void UXGSampleAudioCaptureSubsystem::OnAudioGenerate(const TArray<float>& InAudio, ...)
{
    // 计算 RMS
    float Volume = 0.0;
    for (int32 i = 0; i < InNumSamples; i++)
        Volume += InAudio[i] * InAudio[i];
    Volume = FMath::Sqrt(Volume / InNumSamples);

    // 跨线程传递到游戏线程
    AsyncTask(ENamedThreads::GameThread, [this, Volume]()
    {
        SetVolumeSize(Volume);
    });
}
```

## 跨线程安全性

`OnAudioGenerate` 在**音频线程**中执行，而音量值需要在**游戏线程**被 UI 读取。直接赋值会导致竞态条件。解决方案：

1. `AsyncTask(ENamedThreads::GameThread, ...)` 将音量设置打包到游戏线程执行
2. 音量值通过 `volatile float VolumeSize` 存储（不保证原子性，但单 float 读写在 x64 上是原子的）
3. 蓝图通过 `XGXunFeiGetRealTimeSpeechToTextVolumeSize` 在游戏线程读取

## 蓝图使用

```cpp
float UXGSampleXFLinkBPLibrary::XGXunFeiGetRealTimeSpeechToTextVolumeSize(UObject* WorldContextObject)
{
    UXGSampleAudioCaptureSubsystem* Subsystem = UXGSampleAudioCaptureSubsystem::GetXunFeiAudioCaptureSubsystem();
    return Subsystem ? Subsystem->GetVolumeSize() : 0.0f;
}
```

## 典型应用场景

| 音量范围 | 含义 | 建议 UI 显示 |
|----------|------|------------|
| 0.0 ~ 0.05 | 静音/接近静音 | 红色（检测不到输入） |
| 0.05 ~ 0.2 | 低声说话 | 黄色 |
| 0.2 ~ 0.5 | 正常说话 | 绿色 |
| 0.5 ~ 1.0 | 大声说话/接近爆音 | 红色警告 |

## 注意事项

- RMS 是对数感知的，人耳感受到的音量变化与 RMS 的对数成正比
- 实际 UI 进度条建议使用对数映射以获得更好的视觉体验：`DisplayVolume = FMath::Loge(Volume + 1.0)`
- 音量值每秒更新约 50~100 次（取决于音频帧大小和采样率）
