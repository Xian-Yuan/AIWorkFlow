# 音频消费线程

## 概述

`FXGSampleConsumeVoiceRunnable` 是一个独立后台线程，负责从 `UXGSampleAudioCaptureSubsystem` 的缓冲区轮询音频数据，并通过 `UXGSampleSTTSubsystem` 的 `SendVoiceData` 发送到 WebSocket 服务器。

这是典型的**生产者-消费者模式**：
- **生产者**：`OnAudioGenerate`（音频线程）→ 写入缓冲区
- **消费者**：`ConsumeVoiceRunnable`（独立线程）→ 读取并发送

## FRunnable 实现

```cpp
class FXGSampleConsumeVoiceRunnable : public FRunnable
{
public:
    FXGSampleConsumeVoiceRunnable() : bIsRunning(false) {}

    virtual bool Init() override
    {
        bIsRunning = true;
        return true;
    }

    virtual uint32 Run() override
    {
        while (bIsRunning)
        {
            FPlatformProcess::Sleep(0.01f); // 10ms 轮询间隔

            TArray<float> VoiceData;
            bool bRightVoice = UXGSampleAudioCaptureSubsystem::GetVoiceData(VoiceData);
            if (bRightVoice && VoiceData.Num() > 0)
            {
                UXGSampleSTTSubsystem::RealTimeSTTSubsystemPtr->SendVoiceData(VoiceData);
            }
        }
        return 0;
    }

    virtual void Stop() override
    {
        bIsRunning = false;
    }

    virtual void Exit() override
    {
        UXGSampleAudioCaptureSubsystem::ClearVoiceData();
    }

private:
    volatile bool bIsRunning;
};
```

## 线程创建与启动

在 `UXGSampleSTTSubsystem` 的 `CreateRealTimeSTTSocket` 中（WebSocket 连接成功后）：

```cpp
void UXGSampleSTTSubsystem::CreateRealTimeSTTSocket()
{
    // ... 创建 WebSocket ...

    // 启动音频采集
    UXGSampleAudioCaptureSubsystem::CreateAudioCapture();

    // 启动消费线程
    VoiceConsumeRunnable = new FXGSampleConsumeVoiceRunnable();
    VoiceConsumeRunnableThread = FRunnableThread::Create(
        VoiceConsumeRunnable,
        TEXT("XGSampleConsumeVoiceThread"),
        0, // 默认堆栈大小
        TPri_BelowNormal // 低于正常优先级，避免影响游戏线程
    );
}
```

## 线程销毁

```cpp
void UXGSampleSTTSubsystem::RealeaseVoiceConsumeRunnable()
{
    if (VoiceConsumeRunnable && VoiceConsumeRunnableThread)
    {
        VoiceConsumeRunnable->Stop();     // 设置 bIsRunning = false
        VoiceConsumeRunnableThread->WaitForCompletion(); // 等待线程退出
        delete VoiceConsumeRunnable;      // 安全删除
        VoiceConsumeRunnable = nullptr;
        delete VoiceConsumeRunnableThread;
        VoiceConsumeRunnableThread = nullptr;
    }
}
```

## 线程模型

```
┌──────────────────────┐
│   音频线程 (Audio)    │
│  OnAudioGenerate()   │
│  AppendVoiceData()   │
└────────┬─────────────┘
         │ FScopeLock
         ▼
┌──────────────────────┐
│   VoiceDataSection   │ ← FCriticalSection 保护
│  TArray<float> 缓冲区 │
└────────┬─────────────┘
         │ FScopeLock
         ▼
┌──────────────────────┐
│  消费线程 (Consumer)  │
│  Sleep(10ms)         │
│  GetVoiceData()      │
│  SendVoiceData()     │
└────────┬─────────────┘
         │ Socket->Send
         ▼
┌──────────────────────┐
│  WebSocket 连接       │
│  → 讯飞 STT 服务器    │
└──────────────────────┘
```

## 生产者-消费者同步

| 机制 | 说明 |
|------|------|
| `FCriticalSection` | `VoiceDataSection` 保护缓冲区的读写 |
| `FScopeLock` | 作用域锁，自动 Acquire/Release |
| 10ms 轮询 | 消费者线程每 10ms 检查一次缓冲区 |
| 无条件变量 | 采用轮询而非条件变量唤醒，实现简单但功耗略高 |
