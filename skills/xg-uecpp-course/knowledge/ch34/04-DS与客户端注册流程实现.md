# DS 与客户端注册流程实现

## 工程结构

DS 和客户端共用 `010_XGMultiGame` 工程，同一套代码通过不同的 Target 配置打出两个包（Server 包 + Client 包）。运行时通过 `GetNetMode()` 判断角色。

```

│   │   ├── XGMultiGameGameMode.h/cpp    // DS 端注册触发入口

│   │   ├── XGMultiType.h/cpp             // 消息结构体定义
```

## 架构设计

```
XGMultiGameInstance（核心）
│
├── Init()
│   ├── 获取 UXGWSMClientSubsystem
│   ├── 连接 DSM (127.0.0.1:9033)
│   └── 绑定 4 个连接事件
│
├── OnConnected(ServerConnectionID)
│   ├── 存储 ConnectionID
│   ├── 判断 GetNetMode()
│   │   ├── NM_DedicatedServer → DS 注册
│   │   │   构建 FXGReqInitDSRoleMessage → 序列化 → 发送至 DSM
│   │   └── NM_Client / NM_Standalone → 玩家注册
│   │       构建 FXGReqInitPlayerRoleMessage → 序列化 → 发送至 DSM
│
├── OnMessageReceived(FString, FGuid)
│   │   接收 DSM 回复 → 分发处理
│   ├── RespInitDSRole → 打印"DS 注册成功"
│   ├── RespInitPlayerRole → 打印"Player 注册成功"
│   ├── RespPlayerReqAllDSInfos → 解析 DSList → 打印到屏幕
│   └── RespPlayerCreateNewLevel → 打印"新 DS 已创建"
│
├── Shutdown()
│   ├── 解绑代理
│   └── 断开 WebSocket 连接
│
└── 暴露蓝图调用接口
    ├── RequestAllDSInfo()        // 按键 1 → 获取 DS 列表
    └── RequestCreateNewDS()      // 按键 2 → 请求创建新 DS

XGMultiGameGameMode（DS 层触发）
│
└── BeginPlay()
    └── 绑定 GameInstance.OnConnected
        └── DS 连接成功 → 构建并发送 DSInitInfo
```

## GameInstance 核心实现

[XGMultiGameInstance.h](../../code/010_XGMultiGame/Source/XGMultiGame/XGMultiGameInstance.h) | [XGMultiGameInstance.cpp](../../code/010_XGMultiGame/Source/XGMultiGame/XGMultiGameInstance.cpp)

### Init（初始化 + 连接 DSM）

```cpp
void UXGMultiGameInstance::Init()
{
    Super::Init();

    // 获取 WebSocket 客户端子系统
    ClientSubsystem = GetSubsystem<UXGWSMClientSubsystem>();
    if (!ClientSubsystem) return;

    // 绑定连接事件
    ClientSubsystem->ClientConnectionConnectedSuccessCallBack.AddDynamic(
        this, &UXGMultiGameInstance::OnConnected);
    ClientSubsystem->ClientConnectionConnectedErrorCallBack.AddDynamic(
        this, &UXGMultiGameInstance::OnConnectedFail);
    ClientSubsystem->ClientConnectionReceiveMessageCallBack.AddDynamic(
        this, &UXGMultiGameInstance::OnMessageReceived);
    ClientSubsystem->ClientConnectionClosedCallBack.AddDynamic(
        this, &UXGMultiGameInstance::OnDisconnected);

    // 连接 DSM（IP 和端口由插件默认值决定，DSM 使用 9033）
    ClientSubsystem->LinkSocketServer(TEXT("127.0.0.1"), 9033);
}
```

### Shutdown（清理）

```cpp
void UXGMultiGameInstance::Shutdown()
{
    // 解绑代理
    ClientSubsystem->ClientConnectionConnectedSuccessCallBack.RemoveAll(this);
    ClientSubsystem->ClientConnectionConnectedErrorCallBack.RemoveAll(this);
    ClientSubsystem->ClientConnectionReceiveMessageCallBack.RemoveAll(this);
    ClientSubsystem->ClientConnectionClosedCallBack.RemoveAll(this);

    // 断开连接
    ClientSubsystem->UnLinkAllClientConnection();

    Super::Shutdown();
}
```

### OnConnected（角色判断 + 注册）

连接成功后，根据 `GetNetMode()` 判断当前角色的关键逻辑：

```cpp
void UXGMultiGameInstance::OnConnected(const FGuid& ServerConnectionID)
{
    // 存储 ConnectionID
    ClientConnectionID = ServerConnectionID;

    // 判断角色
    if (GetNetMode() == NM_DedicatedServer)
    {
        // DS 服务器：发送 DS 注册信息
        FXGReqInitDSRoleMessage DSRoleMessage;
        DSRoleMessage.RoleType = EXGMultiRoleType::DS;
        DSRoleMessage.ServerID = FGuid::NewGuid();
        DSRoleMessage.LevelName = TEXT("ThirdPersonMap");

        // 获取 DS 端口（从命令行 -port= 参数解析）
        int32 DSPort = -1;
        FParse::Value(FCommandLine::Get(), TEXT("port="), DSPort);
        DSRoleMessage.ServerPort = DSPort;
        DSRoleMessage.ServerIP = TEXT("127.0.0.1");

        // 双层序列化
        FString DataJson;
        FJsonObjectConverter::UStructToJsonObjectString(DSRoleMessage, DataJson);

        FXGMultiMessage MultiMessage;
        MultiMessage.MessageType = EXGMultiMessageType::ReqInitDSRole;
        MultiMessage.Data = DataJson;

        FString MessageJson;
        FJsonObjectConverter::UStructToJsonObjectString(MultiMessage, MessageJson);

        // 发送
        ClientSubsystem->SendMessageToServer(MessageJson, ClientConnectionID);
    }
    else
    {
        // 客户端玩家：发送玩家注册信息
        FString PlayerName = FString::Printf(TEXT("Player_%d"),
            FMath::RandRange(1000, 9999));

        FXGReqInitPlayerRoleMessage PlayerRoleMessage;
        PlayerRoleMessage.RoleType = EXGMultiRoleType::Player;
        PlayerRoleMessage.PlayerName = PlayerName;

        // 双层序列化（同上模式）
        // ...
        ClientSubsystem->SendMessageToServer(MessageJson, ClientConnectionID);
    }
}
```

#### `GetNetMode()` 返回值说明

| 返回值 | 含义 | 对应包 |
|--------|------|--------|
| `NM_DedicatedServer` | 专用服务器 | Server 包 |
| `NM_Client` | 客户端（已连接） | Client 包 |
| `NM_Standalone` | 单机模式（可能编辑器 PIE） | 编辑器 |
| `NM_ListenServer` | 监听服务器 | 不使用 |

### 接收 DSM 回复

```cpp
void UXGMultiGameInstance::OnMessageReceived(
    const FString& Message, const FGuid& ServerConnectionID)
{
    // 反序列化一级消息
    FXGMultiMessage MultiMessage;
    FJsonObjectConverter::JsonObjectStringToUStruct(Message, &MultiMessage);

    // 按类型处理
    switch (MultiMessage.MessageType)
    {
        case EXGMultiMessageType::RespInitDSRole:
            UE_LOG(LogTemp, Warning, TEXT("注册 DS 成功"));
            break;

        case EXGMultiMessageType::RespInitPlayerRole:
            UE_LOG(LogTemp, Warning, TEXT("玩家注册成功"));
            break;

        case EXGMultiMessageType::RespPlayerReqAllDSInfos:
        {
            // 解析 DS 列表
            FXGRespPlayerAllDSInfosInfo RespAllDSInfos;
            FJsonObjectConverter::JsonObjectStringToUStruct(
                MultiMessage.Data, &RespAllDSInfos);
            // 打印到屏幕
            for (const auto& DSInfo : RespAllDSInfos.DSList)
            {
                FString Message = FString::Printf(TEXT(
                    "DS: %s:%d - %s"),
                    *DSInfo.ServerIP, DSInfo.ServerPort, *DSInfo.LevelName);
                // 屏幕打印
                GEngine->AddOnScreenDebugMessage(-1, 15.0f,
                    FColor::Green, Message);
            }
            break;
        }

        case EXGMultiMessageType::RespPlayerCreateNewLevel:
            UE_LOG(LogTemp, Warning, TEXT("新 DS 已创建"));
            break;
    }
}
```

## GameMode 绑定（DS 注册触发）

[GameMode.h](../../code/010_XGMultiGame/Source/XGMultiGame/XGMultiGameGameMode.h) | [GameMode.cpp](../../code/010_XGMultiGame/Source/XGMultiGame/XGMultiGameGameMode.cpp)

`XGMultiGameGameMode` 在 BeginPlay 中绑定 GameInstance 的 `OnConnected` 委托，获取 DS 端口信息：

```cpp
void AXGMultiGameGameMode::BeginPlay()
{
    Super::BeginPlay();

    // 获取 GameInstance
    UXGMultiGameInstance* GameInstance = GetGameInstance<UXGMultiGameInstance>();

    // 绑定连接成功事件
    GameInstance->OnConnected.AddDynamic(
        this, &AXGMultiGameGameMode::OnConnectionSuccess);
}

void AXGMultiGameGameMode::EndPlay(const EEndPlayReason::Type EndPlayReason)
{
    if (GameInstance)
    {
        GameInstance->OnConnected.RemoveAll(this);
    }
    Super::EndPlay(EndPlayReason);
}
```

## 已知问题

### 时序 BUG

`LinkSocketServer` 的 `ConnectSuccess` 回调触发时机可能早于事件绑定，导致 `OnConnected` 永远不会被调用，自动注册流程不触发。

**表现**：DS/Player 启动后，DSM 收不到任何注册消息。

**临时修复**：不在 `Init()` 中立即连接，而是在第一个 Tick 或延迟帧后，等事件系统准备好再连接。

## 关键注册流程图

```

  │
  ├── Init()
  │   ├── 获取 ClientSubsystem
  │   ├── 绑定 4 个事件 (OnConnected/OnConnectFail/OnMessageReceived/OnDisconnected)
  │   └── LinkSocketServer(127.0.0.1, 9033)
  │
  ├── OnConnected(ConnectionID)  ← 时序问题：可能早于绑定
  │   ├── GetNetMode() == NM_DedicatedServer
  │   ├── FParse::Value("-port=", DSPort)
  │   ├── 构建 FXGReqInitDSRoleMessage (Port, IP, LevelName, ServerID)
  │   └── 双层 JSON 序列化 → SendMessageToServer
  │
  └── OnMessageReceived(RespInitDSRole)
      └── DS 注册成功 → 等待玩家接入

  │
  ├── Init()
  │   ├── 获取 ClientSubsystem
  │   ├── 绑定 4 个事件
  │   └── LinkSocketServer(127.0.0.1, 9033)
  │
  ├── OnConnected(ConnectionID)  ← 时序问题：可能早于绑定
  │   ├── GetNetMode() != NM_DedicatedServer
  │   ├── 构建 FXGReqInitPlayerRoleMessage (PlayerName)
  │   └── 双层 JSON 序列化 → SendMessageToServer
  │
  ├── OnMessageReceived(RespInitPlayerRole)
  │   └── 玩家注册成功
  │
  ├── RequestAllDSInfo()  ← 用户按键 1
  │   └── OnMessageReceived(RespPlayerReqAllDSInfos)
  │       └── 打印 DS 列表到屏幕
  │
  └── RequestCreateNewDS()  ← 用户按键 2
      └── OnMessageReceived(RespPlayerCreateNewLevel)
          └── 新 DS 已创建
```
