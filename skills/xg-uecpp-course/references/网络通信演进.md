# 网络通信演进

## 8 章演进全景

从第 25 章到第 32 章，课程沿一条通信链路逐层深入。伴随协议升级，异步架构、认证方式、工程组织也在同步演进：

| 章节 | 案例 | 传输协议 | 异步模式 | 认证方式 | 代码组织 |
|------|------|---------|---------|---------|---------|
| 25 | HTTP 获取公网时间 | HTTP GET | UBlueprintAsyncActionBase | 无 | 单个 C++ 文件 |
| 26 | HTTP 登录注册 | HTTP POST | 独立程序主循环 | MD5 令牌 | 独立程序 + 插件 |
| 27 | 摄像头活体检测 | HTTP POST Upload | UBlueprintAsyncActionBase | HMAC-SHA256 | 双插件分离 |
| 28 | 百度文心一言 | HTTP Stream | AsyncAction + 流式回调 | HMAC-SHA256 (BCE v1) | 独立 Link 插件 |
| 29 | 讯飞语音识别 | WebSocket | Subsystem + FRunnable | HMAC-SHA1 | 插件内多子系统 |
| 30 | 讯飞语音合成 | WebSocket | UBlueprintAsyncActionBase | HMAC-SHA256 | 同插件 TTS 模块 |
| 31 | MMO 分布式通信 | WebSocket Client+Server | Subsystem + Tick | 双 GUID | 自建 WSM 插件 |
| 32 | TCP 邮箱验证码 | TCP + SMTP | TSharedFromThis + FRunnable | AUTH LOGIN (Base64) | 独立 Email 插件 |

## 协议演进：HTTP → WebSocket → TCP

| 维度 | HTTP | WebSocket | TCP |
|------|------|-----------|-----|
| 模型 | 请求-响应 | 全双工持久连接 | 面向流字节通信 |
| 连接 | 每次请求新建 | 一次握手持久化 | 建立后持续 |
| 协议框架 | FHttpModule 原生支持 | FWebSocketsModule 原生支持 | FSocket 裸操作 |
| 服务端框架 | FHttpServerModule | IWebSocketServer（WebSocketNetworking） | 纯 Socket |
| 状态管理 | 无状态 | 连接状态 | 应用层状态机 |
| 消息边界 | HTTP 协议自带 | Frame 帧 | 需要应用层解析 |
| 消息解析 | FJsonObjectConverter | FJsonObjectConverter | 手动协议解析 |

## 认证方式演进

| 认证方式 | 所用章节 | 原理 |
|---------|---------|------|
| 无认证 | 25 | 公开 API |
| MD5 令牌 | 26 | 密码+时间戳 MD5 链式哈希 |
| HMAC-SHA256 | 27-28 | 请求体 + 密钥 HMAC 签名（BCE v1） |
| HMAC-SHA1 | 29 | 语音识别 WebSocket URL 签名 |
| HMAC-SHA256 | 30 | 语音合成 URL 签名 |
| 双 GUID（Request/Response） | 31 | 消息匹配 + 连接验证 |
| AUTH LOGIN (Base64) | 32 | SMTP 协议标准认证 |

## 核心链路

### 链路一：HTTP 请求

```cpp
FHttpRequestRef Request = FHttpModule::Get().CreateRequest();
Request->SetURL(TEXT("https://api.example.com/data"));
Request->SetVerb(TEXT("POST"));
Request->SetHeader(TEXT("Content-Type"), TEXT("application/json"));
Request->SetContentAsString(JsonBody);
Request->OnProcessRequestComplete().BindUObject(this, &UMyClass::OnResponse);
Request->ProcessRequest();
```

### 链路二：WebSocket 双向通信

```cpp
TSharedPtr<IWebSocket> WS = FWebSocketsModule::Get().CreateWebSocket(URL);
WS->OnConnected().AddLambda([](){ /* 连接成功 */ });
WS->OnMessage().AddLambda([](const FString& Msg){ /* 收到消息 */ });
WS->OnConnectionError().AddLambda([](const FString& Err){ /* 连接错误 */ });
WS->OnClosed().AddLambda([](int32 Code, const FString& Reason){ /* 关闭 */ });
WS->Connect();
WS->Send(TEXT("Hello"));
```

### 链路三：TCP 字节流

```cpp
FSocket* Socket = FTcpSocketBuilder(TEXT("MySocket")).AsBlocking().Build();
TSharedRef<FInternetAddr> Addr = ISocketSubsystem::Get(PLATFORM_SOCKETSUBSYSTEM)->CreateInternetAddr();
Addr->SetIp(*ServerIP);
Addr->SetPort(ServerPort);
Socket->Connect(*Addr);
// 发送/接收字节流
Socket->Send(SendBuf, SendSize, BytesSent);
Socket->Recv(RecvBuf, RecvBufSize, BytesRead);
```

### 链路四：SMTP 状态机

```
220 → EHLO → 250 → AUTH LOGIN → 334 → 
Base64(Username) → 334 → Base64(Password) → 235 → 
MAIL FROM → 250 → RCPT TO → 250 → 
DATA → 354 → 邮件 + "\r\n.\r\n" → 250 → 
QUIT → 221
```
