# 字符串处理详解

## 概述

UE 提供三套字符串类型，各自面向不同场景：FString（可变字符串，数据载具）、FName（不可变索引，资产/标识符）、FText（本地化文本，UI 显示）。底层字符编码为 UTF-16（TCHAR），外部接口支持多种编码转换。

## 三种字符串类型对比

| 类型 | 特性 | 使用场景 |
|------|------|---------|
| **FString** | 可变、堆分配、支持修改操作 | 文件路径、网络数据传输、拼装内容 |
| **FName** | 不可变、全局索引池、相等比较极快（整数比较） | 资产引用、Tag、键名、动画插槽 |
| **FText** | 可本地化、支持格式化和复数规则 | UI 文本、面向用户显示 |

```cpp
FString Str = TEXT("Hello");
FName Name = FName("PlayerCollisionProfile");
FText Text = FText::FromString(TEXT("Hello"));
```

## TCHAR 与编码体系

| 符号 | 含义 |
|------|------|
| `TCHAR` | 项目字符类型（UE5 默认 `wchar_t`，UTF-16） |
| `TEXT()` | 字符串字面量包装宏，自动适配 TCHAR |
| `FTCHARToUTF8` | TCHAR → UTF-8 转换 |
| `FUTF8ToTCHAR` | UTF-8 → TCHAR 转换 |

```cpp
// 编码转换
FTCHARToUTF8 Converter(MyFString);
const char* Utf8Str = Converter.Get();
```

## FString 基本操作

### 创建与初始化

```cpp
FString Empty;
FString FromLiteral = TEXT("Hello");
FString FromPrintf = FString::Printf(TEXT("Count: %d, Name: %s"), 42, *NameStr);
FString FromFormat = FString::Format(TEXT("{0} has {1} items"), { ItemName, Count });
```

### 查询与比较

```cpp
Str.Len();                    // 字符长度
Str.IsEmpty();                // 是否为空
Str.Equals(Other, ESearchCase::IgnoreCase);  // 忽略大小写比较
Str == Other;                 // 大小写敏感比较
Str.StartsWith(TEXT("Prefix"));  // 前缀判断
Str.Contains(TEXT("Sub"));    // 包含判断
```

### 分割与拼接

```cpp
// 分割
TArray<FString> Parts;
Str.ParseIntoArray(Parts, TEXT(","), true);  // 按逗号分割
Str.ParseIntoArrayLines(Parts, true);       // 按换行分割

// 拼接
FString Joined = FString::Join(Parts, TEXT(","));
```

### 替换与裁剪

```cpp
Str.Replace(TEXT("old"), TEXT("new"));           // 全部替换
Str.ReplaceInline(TEXT("old"), TEXT("new"));     // 就地替换
Str.TrimStartAndEnd();                           // 去除首尾空白
Str.RemoveFromStart(TEXT("Prefix"));             // 移除前缀
```

## FString 类型转换

```cpp
// 数值 ↔ 字符串
int32 Val = FCString::Atoi(*Str);
float ValF = FCString::Atof(*Str);
FString NumStr = FString::FromInt(42);
FString FloatStr = FString::SanitizeFloat(3.14f);

// FName ↔ FString
FName Name = FName(*Str);
FString FromName = Name.ToString();

// FText ↔ FString
FText Text = FText::FromString(Str);
FString FromText = Text.ToString();

// std::string ↔ FString
std::string StdStr = TCHAR_TO_UTF8(*FString);
FString FromStd = FString(UTF8_TO_TCHAR(StdStr.c_str()));
```

## 已弃用的编码转换宏

以下宏在 UE5 中已弃用，应使用上方的显式转换类：

```cpp
// ❌ 已弃用
TCHAR_TO_ANSI, ANSI_TO_TCHAR  // 不推荐
```

## FText 本地化与格式化

```cpp
// 字面量本地化
FText::FromString(TEXT("Hello"));               // 不参与本地化
NSLOCTEXT("Category", "Key", "Hello");          // 参与本地化
LOCTEXT("Key", "Hello");                        // NSLOCTEXT 单文件简化版（文件顶部声明）

// 带参数格式化
FFormatOrderArguments Args;
Args.Add(0, FText::FromString(TEXT("World")));
Args.Add(1, 42);
FText::Format(LOCTEXT("HelloKey", "Hello {0}, count: {1}"), Args);
```

## FName 使用详解

```cpp
// 创建（自动注册到全局池）
FName Name1 = FName(TEXT("CollisionProfile"));
FName Name2 = FName(TEXT("CollisionProfile"), FNAME_Find);  // 仅查找，不创建新条目

// 比较（整数比较，极快）
if (Name1 == Name2) { }  // 最常用

// 数值后缀
FName WithNumber = FName(TEXT("Name"), 3);  // "Name_3"
```

## 三者转换关系

```
FString ←→ FName （通过 FName(*Str) / Name.ToString()）
FString ←→ FText （通过 FText::FromString() / Text.ToString()）
FText  ←x→ FName （不推荐直接转换）
```
