# 容器选型指南

## 决策树

```
需要有序序列？
├── 是 → TArray（默认容器）
└── 否 → 需要键值映射？
           ├── 是 → TMap
           └── 否 → 仅需集合存在性？
                      ├── 是 → TSet
                      └── 否 → TArray（默认）
```

## 核心特性对比

| 维度 | TArray | TMap | TSet |
|------|--------|------|------|
| 底层结构 | 动态数组（连续内存） | 哈希表 + 桶 | 哈希表 + 桶 |
| 元素形式 | 单一类型 | TPair<Key, Value> | 单一类型 |
| 顺序 | 有序（插入序） | 无序 | 无序 |
| 随机访问 | O(1) 索引 | O(1) 键查值 | O(1) 元素查找 |
| 插入 | O(1) 末尾 / O(n) 中间 | O(1) 平均 | O(1) 平均 |
| 查找 | O(n) | O(1) 键 / O(n) 反查 | O(1) |
| 移除 | O(n) | O(1) 平均 | O(1) 平均 |
| 元素唯一性 | 否（AddUnique 可去重） | 键唯一 | 元素唯一 |
| 扩容 | 首次 4，之后每次 +16 | Rehash | Rehash |
| 指针稳定性 | 扩容时失效 | 插入/移除后可能失效 | 插入/移除后可能失效 |
| 使用频率 | 最高 | 中 | 低 |

## 添加元素

| 操作 | TArray | TMap | TSet |
|------|--------|------|------|
| 添加 | `Add(Val)` | `Add(Key, Val)` | `Add(Val)` |
| 原地构造 | `Emplace(Args...)` | `Emplace(Key, Val)` | `Emplace(Args...)` |
| 合并 | `Append(Other)` | `Append(Other)` | `Append(Other)` |
| 去重添加 | `AddUnique(Val)` | 默认唯一 | 默认唯一 |
| 默认构造 | `AddDefaulted()` | — | — |
| 指定位置 | `Insert(Val, Index)` | — | — |

## 查询元素

| 操作 | TArray | TMap | TSet |
|------|--------|------|------|
| 存在性 | `Contains(Val)` | `Contains(Key)` | `Contains(Val)` |
| 查找 | `Find(Val)` 返回索引 | `Find(Key)` 返回值指针 | `Find(Val)` 返回元素指针 |
| 获取引用 | `operator[]` | `operator[]`（键不存在则崩溃） | `operator[FSetElementId]` |
| 默认值 | — | `FindRef(Key)` 返回副本 | — |
| 不存在时创建 | — | `FindOrAdd(Key)` | — |
| 谓词查找 | `FindByPredicate(Pred)` | — | — |
| 反向查找 | — | `FindKey(Val)` O(n) | — |

## 移除元素

| 操作 | TArray | TMap | TSet |
|------|--------|------|------|
| 按值/键 | `Remove(Val)` | `Remove(Key)` | `Remove(Val)` |
| 按索引 | `RemoveAt(Index)` | — | `Remove(FSetElementId)` |
| 谓词 | `RemoveAll(Pred)` | — | — |
| 全部清除 | `Empty()` / `Reset()` | `Empty()` / `Reset()` | `Empty()` / `Reset()` |
| 快速移除（不保序） | `RemoveAtSwap(Index)` | — | — |

## 迭代方式

```cpp
// for-range（推荐）
for (const auto& Elem : Array) { }
for (const auto& Elem : Map) { Elem.Key; Elem.Value; }
for (const auto& Elem : Set) { }

// 迭代器
for (auto It = Container.CreateConstIterator(); It; ++It) { }
```

### 迭代中安全移除

- **TArray**：反向遍历（从后往前）
- **TMap/TSet**：`It.RemoveCurrent()`（CreateIterator 模式）

## 自定义类型要求

| 容器 | 需要重载 |
|------|---------|
| TArray | `operator==`（查找/比较），排序需要 `operator<` |
| TMap（键） | `operator==` + `GetTypeHash()` |
| TSet（元素） | `operator==` + `GetTypeHash()` |

## 选型建议

| 需求 | 推荐容器 |
|------|---------|
| 按索引访问、保持顺序 | TArray |
| 按键快速查找值 | TMap |
| 快速集合存在性检查 | TSet |
| 不确定、通用场景 | TArray |
| 频繁中间插入 | 考虑 TArray 反序操作 + Add，或无序用 Swap 移除 |
| 大量元素 + 频繁查找 | TMap/TSet |
| 小量元素（< 50） | TArray 线性查找即可 |
| 需要排序输出 | TArray |

## 容器转换

```cpp
TSet → TArray:   Set.Array()
TMap → TArray:   Map.GenerateKeyArray(OutArr) / GenerateValueArray(OutArr)
TArray → TSet:   TSet<T>(Array)
TArray → TMap:   逐元素 Map.Add()
```
