# 委托体系详解

## 四种委托类型

| 委托类型 | 声明宏前缀 | 绑定方式 | 广播方式 | 蓝图可视 | 可序列化 |
|---------|-----------|---------|---------|---------|---------|
| 单播 Native | `DECLARE_DELEGATE` | Bind | Execute/ExecuteIfBound | ✗ | ✗ |
| 多播 Native | `DECLARE_MULTICAST_DELEGATE` | Add | Broadcast | ✗ | ✗ |
| 单播 Dynamic | `DECLARE_DYNAMIC_DELEGATE` | Bind | Execute/ExecuteIfBound | ✓ | ✓ |
| 多播 Dynamic | `DECLARE_DYNAMIC_MULTICAST_DELEGATE` | Add/BindEvent | Broadcast | ✓（BlueprintAssignable） | ✓ |

## 声明宏体系

```cpp
// 无参数
DECLARE_DELEGATE(FMyDelegate)
DECLARE_MULTICAST_DELEGATE(FMyMulticastDelegate)
DECLARE_DYNAMIC_DELEGATE(FMyDynamicDelegate)
DECLARE_DYNAMIC_MULTICAST_DELEGATE(FMyDynamicMulticastDelegate)

// 带参数（最多支持 8 个 + 返回值）
DECLARE_DELEGATE_OneParam(FMyDelegate, int32)
DECLARE_DELEGATE_TwoParams(FMyDelegate, int32, FString)
DECLARE_DELEGATE_RetVal_OneParam(int32, FMyDelegate, FString)

// 多播不可有返回值，动态委托参数必须支持反射
```

## 单播委托绑定方式

| 绑定方式 | 语法 | 生命周期安全 | 适用场景 |
|---------|------|-------------|---------|
| BindUObject | `BindUObject(this, &Fcn)` | 自动（UObject 销毁自动断开） | UObject 成员 |
| BindRaw | `BindRaw(this, &Fcn)` | 手动 | 原生 C++ 对象 |
| BindSP | `BindSP(SharedPtr, &Fcn)` | 自动（TSharedPtr 弱引用） | TSharedPtr 对象 |
| BindStatic | `BindStatic(&StaticFcn)` | N/A | 全局静态函数 |
| BindLambda | `BindLambda([this](){})` | 手动捕获 | 临时回调 |
| BindThreadSafeSP | `BindThreadSafeSP(SharedPtr, &Fcn)` | 线程安全 | 多线程环境 |
| BindUFunction | `BindUFunction(this, FName)` | 自动 | UFUNCTION |
| BindWeakLambda | `BindWeakLambda(this, [this]{})` | 自动（UDestructibleWeakPtr） | Lambda + 安全 |

## Payload 负载参数

绑定时注入额外数据，运行时无需额外参数：

```cpp
DECLARE_DELEGATE_OneParam(FOnComplete, int32);

// 绑定时注入 FString
FOnComplete Delegate;
Delegate.BindLambda([](int32 Code) { /* 运行时处理 */ }, TEXT("extra_data"));
// 实际上需要用 BindRaw 或 BindSP 的 Payload 重载
// 更常见的是用 Lambda 捕获
Delegate.BindLambda([ExtraData](int32 Code) { /* 使用 ExtraData */ });
```

## 多播委托管理

```cpp
FMyMulticastDelegate Delegate;

// 添加
FDelegateHandle Handle = Delegate.AddUObject(this, &Fcn);

// 广播（触发所有绑定）
Delegate.Broadcast();

// 定点移除（推荐）
Delegate.Remove(Handle);

// 清除所有
Delegate.Clear();
```

## 动态委托蓝图集成

### BlueprintAssignable

```cpp
UPROPERTY(BlueprintAssignable)
FOnResponse OnResponse;  // 声明为 UPROPERTY

// C++ 广播
OnResponse.Broadcast(Result);

// 蓝图节点中显示为输出执行引脚
```

### Bind Event（蓝图中替代 Add）

```
蓝图中使用 Bind Event 节点绑定动态多播委托

```

### AddDynamic / RemoveDynamic

```cpp
// C++
AddDynamic(this, &UMyClass::OnResponse);
RemoveDynamic(this, &UMyClass::OnResponse);

// 蓝图中 AddDynamic 节点自动处理
```

## Create 方法 + 委托替换

```cpp
DECLARE_DELEGATE(FSimpleDelegate);

FSimpleDelegate Delegate1, Delegate2;

Delegate1.BindLambda([]() { UE_LOG(LogTemp, Log, TEXT("Delegate1")); });
Delegate2.BindLambda([]() { UE_LOG(LogTemp, Log, TEXT("Delegate2")); });

// 运行时替换绑定
Delegate1 = Delegate2;  // CreateCopyIfBound

// 或使用 Create 工厂
FSimpleDelegate NewDel = FSimpleDelegate::CreateLambda([](){});
```

## 使用规范

| 场景 | 推荐委托类型 |
|------|-------------|
| C++ 内部回调（一对一） | 单播 Native |
| C++ 事件系统（一对多） | 多播 Native |
| 蓝图事件分发器 | 动态多播（BlueprintAssignable） |
| UFUNCTION 绑定 | BindUFunction / AddDynamic |
| 线程安全 | BindThreadSafeSP / FThreadSafeDelegate |
| 无返回值轻量 | DECLARE_DELEGATE |
| 需要返回值 | DECLARE_DELEGATE_RetVal |
