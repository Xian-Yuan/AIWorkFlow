# TD（俯视角）模板概述

## 架构特点

TD 模板与 FP/TP 不同，核心交互为**鼠标点击移动**而非 WASD 键盘移动。

| 方面 | 说明 |
|------|------|
| 摄像机 | CameraBoom 固定俯视角，Camera 俯视场景 |
| 角色旋转 | `bUseControllerRotationYaw = false` + `bOrientRotationToMovement = true`（移动跟随转向） |
| 核心输入 | 单个 `IA_Click` Input Action（Boolean 类型） |
| 移动方式 | 鼠标点击目标位置 → AI 寻路移动 / 鼠标拖拽方向 → 持续移动 |

## 鼠标点击移动两种模式

### 1. 持续移动模式（Triggered：按住鼠标）

PlayerController 每帧执行 LineTrace 检测鼠标下的 HitResult，计算角色朝向光标的方向向量，调用 `AddMovementInput` 持续移动。

### 2. 点击移动模式（Completed/Canceled：松开鼠标）

检测按下到松开的耗时，若耗时短（单击而非长按），调用：
```cpp
UAIBlueprintHelperLibrary::SimpleMoveToLocation(GetPawn(), Hit.Location);
```
同时生成 Niagara 粒子特效作为点击位置的标记。

## PlayerController 关键设置

```cpp
APlayerController::BeginPlay()
{
    Super::BeginPlay();
    bShowMouseCursor = true;                    // 显示鼠标光标
    // 注册 IMC
    Subsystem->AddMappingContext(IMC_Default, 0);
}

// 输入绑定
EnhancedInputComponent->BindAction(ClickAction, ETriggerEvent::Triggered, this, &ATDPlayerController::OnClickTriggered);
EnhancedInputComponent->BindAction(ClickAction, ETriggerEvent::Completed, this, &ATDPlayerController::OnClickCompleted);
EnhancedInputComponent->BindAction(ClickAction, ETriggerEvent::Canceled,  this, &ATDPlayerController::OnClickCanceled);
```

## 构建依赖

C++ 项目需要额外添加模块依赖：
| Build.cs 模块 | 用途 |
|---|---|
| `"AIModule"` | `UAIBlueprintHelperLibrary` |
| `"NiagaraSystem"` | 点击位置粒子特效 |

## 角色移动设置

- `bUseControllerRotationYaw = false`：角色不跟随控制器旋转
- `bOrientRotationToMovement = true`：角色面朝移动方向自动旋转
- CameraBoom 固定角度俯视，不跟随角色旋转
