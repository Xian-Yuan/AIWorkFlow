# 第九章：委托

委托是 UE 中**类型安全的函数指针**，用于解耦调用者和响应者的关系。本章覆盖 UE 全部四种委托类型：Native 单播、Native 多播、动态单播、动态多播。

## 核心内容

- 四种委托类型的声明宏体系、API 模式、生命周期管理
- 单播委托的多种绑定方式（UObject/Raw/SP/Lambda/Static/ThreadSafeSP/UFunction/WeakLambda）
- Payload 负载参数机制——绑定时注入额外数据
- Create 方法 + 委托替换技术（运行时动态切换回调）
- 多播委托的一对多广播机制与 FDelegateHandle 定点移除
- 动态委托的蓝图集成（BlueprintAssignable、Bind Event、AddDynamic/RemoveDynamic）
