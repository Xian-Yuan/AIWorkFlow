# 客户端实现与全流程通讯

## 架构总览

```
UXGWSMClientSubsystem（UGameInstanceSubsystem + FTickableGameObject）
  │
  ├── ClientConnections（TMap<FGuid, FXGWSMClientConnection>）
  │     ├── Connection_1（TSharedPtr<IWebSocket> + 双GUID + 状态 + 3s心跳）
  │     ├── Connection_2
  │     └── Connection_N
  │
  ├── ClientConnectionDatas（TMap<FGuid, TArray<uint8>>）
  │     └── 碎片数据累积缓冲区（按连接 ID 索引）
  │
  ├── 4 个蓝图广播委托
  │   ├── ClientConnectionConnectedSuccessCallBack
  │   ├── ClientConnectionConnectedErrorCallBack
  │   ├── ClientConnectionReceiveMessageCallBack
  │   └── ClientConnectionClosedCallBack
  │
  └── Tick(float DeltaTime)
        └── TickClientConnection()
              └── 遍历所有连接 → Tick() → 超时检测 → 释放
```

## 客户端连接创建

### LinkSocketServer

```cpp
void UXGWSMClientSubsystem::LinkSocketServer(
    const FString& InServerURL,
    FGuid& OutClientConnectionID)
```

创建连接流程：

```
1. 生成 FGuid（OutClientConnectionID）
   → 作为该连接在本地的唯一标识

2. 创建 FXGWSMClientConnection
   → 将 GUID 存入连接结构

3. 存入 ClientConnections TMap
   
4. 异步创建 WebSocket（AsyncTask → GameThread）
   → CreateSocket(InServerURL, protocol, headers, GUID)
```

### CreateSocket

```cpp
void UXGWSMClientSubsystem::CreateSocket(
    const FString& InServerURL,
    const FString& InServerProtocol,
    const TMap<FString, FString>& InServerUpgradeHeaders,
    const FGuid InTargetClientConnectionID)
```

创建与配置流程：

```
1. FModuleManager::LoadModuleChecked("WebSockets")
   → 确保 WebSockets 模块已加载

2. FWebSocketsModule::Get().CreateWebSocket(URL, Protocol)
   → 创建 TSharedPtr<IWebSocket>

3. 绑定 6 个回调：
   ├── OnConnected      → 连接成功
   ├── OnConnectionError → 连接失败
   ├── OnClosed         → 连接关闭
   ├── OnMessage        → 文本消息（未使用，使用二进制消息）
   ├── OnMessageSent    → 消息发送确认（仅日志）
   └── OnBinaryMessage  → ★ 二进制消息（主要通信通道）

4. WebSocket->Connect()
   → 发起连接
```

### 6 个回调的作用

| 回调 | 触发时机 | 处理逻辑 |
|------|---------|---------|
| `OnConnected` | WebSocket 连接建立成功 | 调用 `CallInitServerConnection` 发送初始化消息 |
| `OnConnectionError` | 连接失败（网络/超时/拒绝） | 广播失败委托，清理资源 |
| `OnClosed` | 连接被关闭 | 设置状态为 Quit，广播关闭委托，清理资源 |
| `OnMessage` | 收到文本消息 | 本项目中不使用（空实现） |
| `OnMessageSent` | 消息发送完成 | 仅日志输出 |
| `OnBinaryMessage` | **收到二进制消息** | **主通信通道**：数据重组 + 解码 + 分发 |

## 连接状态机（客户端）

```
初始状态
    │
    ▼
┌────────────┐
│    None    │ ← CreateSocket 后、Connect 前
└──────┬─────┘
       │ OnConnected
       ▼
┌─────────────────────┐
│ InitServerConnection │ ← 发送 ClientCallServerInit 后
└──────┬──────────────┘
       │ 收到 ServerResponseClientInit
       ▼
┌──────────┐
│   Tick   │ ← 正常通信状态（发送/接收消息 + 3s 心跳）
└─────┬────┘
      │
      ├── 主动请求断开 → Send ClientReqQuit → PrepareToQuit
      ├── 收到 ServerReqQuit → Send ClientAllowServerQuit → Quit
      ├── 10s 超时未收到心跳响应 → OutOfTime（清理）
      │
      ▼
┌──────────────┐
│ PrepareToQuit │ ← 已发送断开请求，等待服务器确认
└──────┬───────┘
       │ 收到 ServerAllowClientQuit
       ▼
┌──────────┐
│   Quit   │ → ClearPointedClientConnectionResources
└──────────┘

超时路径：
  Tick → IsOutOfTime == true → OutOfTime
    → 直接清理资源（不经过握手关闭）
```

## 消息发送

### Send（在 FXGWSMClientConnection 中）

```cpp
void FXGWSMClientConnection::Send(const FXGWSMMessage& InMessage)
```

发送流程：

```
FXGWSMMessage
  → FJsonObjectConverter::UStructToJsonObjectString（结构体 → JSON 字符串）
  → UXGWSMUtilBPLibrary::XGMessageEncode（JSON → Base64 → TArray<uint8>）
  → ClientWebSocket->Send(InData.GetData(), InData.Num(), false)（二进制发送）
```

## 消息接收与处理

### OnBinaryMessage - 数据重组

二进制消息可能被拆分为多个 Fragment（分片），需要重组：

```cpp
void UXGWSMClientSubsystem::OnBinaryMessage(
    const void* Data, SIZE_T Size, bool bIsLastFragment,
    FGuid InTargetClientConnectionID)
```

重组逻辑：

```
收到二进制片段
  │
  ├── bIsLastFragment == false（中间片段）
  │     └── 追加到 ClientConnectionDatas[ConnectionID] 累积缓冲区
  │
  └── bIsLastFragment == true（最后一个片段）
        ├── 如果有累积数据 → 追加后解码
        ├── 如果没有累积数据 → 直接解码
        └── 解码后解析 JSON → OnMessageDispatch
```

### OnMessageDispatch - 5 种消息分发

```cpp
void UXGWSMClientSubsystem::OnMessageDispatch(
    const FGuid& InTargetClientConnectionID,
    const FXGWSMMessage& InMessage)
```

根据 ActionType 分发到 5 个处理函数：

| ActionType | 处理函数 | 逻辑 |
|-----------|---------|------|
| `ServerResponseClientInit` | `OnServerResponseClientInit` | 保存 ServerConnectionID，状态→Tick，广播连接成功委托 |
| `ServerResponseTick` | `OnServerResponseTick` | 调用 `UpdateIntervalTime()` 更新心跳计时 |
| `ServerMessageToClient` | `OnServerMessageToClient` | 广播接收消息委托到蓝图 |
| `ServerReqQuit` | `OnServerReqQuit` | 状态→Quit，回复 `ClientAllowServerQuit` |
| `ServerAllowClientQuit` | `OnServerAllowClientQuit` | 状态→Quit，清理资源 |

## 心跳机制

### 客户端心跳发送

```cpp
// FXGWSMClientConnection::Tick(float InDeltaTime)
void FXGWSMClientConnection::Tick(float InDeltaTime)
{
    if (ClientConnectionStatus == EXGWSMClientConnectionStatus::Tick)
    {
        AccumulateIntervalTime += InDeltaTime;

        if (AccumulateIntervalTime >= IntervalTime)  // IntervalTime = 3.0f
        {
            // 发送心跳消息
            SendHeartBeat();
            
            // 重置计时
            AccumulateIntervalTime = 0.0f;
        }
    }
}
```

### 心跳超时检测

```cpp
// FXGWSMClientConnection::IsOutOfTime()
bool FXGWSMClientConnection::IsOutOfTime()
{
    if (ClientConnectionStatus == EXGWSMClientConnectionStatus::Tick)
    {
        return AccumulateIntervalTime >= OutOfTimeThreshold;  // 10.0f
    }
    return false;
}
```

### 心跳时间参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 心跳间隔（IntervalTime） | 3s | 客户端每 3s 发送一次心跳 |
| 超时阈值（OutOfTimeThreshold） | 10s | 超过 10s 未收到服务端响应判断超时 |
| 心跳消息 | `ClientCallTick` Action | 服务端收到后回复 `ServerResponseTick` |

### UpdateIntervalTime

```cpp
void UpdateIntervalTime()
{
    // 重置累计时间（收到服务端心跳响应时调用）
    AccumulateIntervalTime = 0.0f;
}
```

## 连接资源清理

| 清理方法 | 作用范围 | 调用时机 |
|---------|---------|---------|
| `ClearPointedClientConnectionResources` | 单个连接 | 关闭/超时/错误时 |
| `ClearAllClientConnectionResources` | 全部连接 | Subsystem Deinitialize |

清理内容：
```
ClientConnections.Remove(ConnectionID)         // 移除连接
ClientConnectionDatas.Remove(ConnectionID)      // 移除碎片数据缓冲区
```

## 客户端 API 总览

| 方法 | 参数 | 说明 |
|------|------|------|
| `LinkSocketServer` | `URL, OUT ClientConnectionID` | 创建新连接 |
| `UnLinkPointedClientConnection` | `TargetClientConnectionID` | 断开指定连接（发送 ClientReqQuit） |
| `UnLinkAllClientConnection` | 无 | 断开所有连接 |
| `SendMessageToServer` | `Message, TargetClientConnectionID` | 发送业务消息 |
| `ClearPointedClientConnectionResources` | `TargetClientConnectionID` | 清理连接资源 |
| `ClearAllClientConnectionResources` | 无 | 清理全部连接资源 |
