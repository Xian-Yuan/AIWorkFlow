# 空白插件与内容插件与蓝图函数库

## 概述

三种基础插件类型，复杂度从低到高：Blank（仅有代码骨架）、Content Only（仅有蓝图资源）、Blueprint Function Library（静态函数导出）。

---

## Blank 空插件

### 特点

- 最简化的插件代码骨架
- 没有可视化界面，没有自动生成的菜单/工具栏按钮
- 适合作为其他插件的起点

### .uplugin 配置

[XGBlank.uplugin](file:///d:/UPS/GitLab/XGUnrealNote/Courses/UE-CPP-Basic-to-Advanced/code/002_XGSampleTemp/Plugins/XGBlank/XGBlank.uplugin)：

```json
{
    "FileVersion": 3,
    "Modules": [
        {
            "Name": "XGBlank",
            "Type": "Runtime",
            "LoadingPhase": "Default"
        }
    ]
}
```

### 模块生命周期

模块类继承 `IModuleInterface`，实现两个虚函数：

| 方法 | 触发时机 | 用途 |
|------|---------|------|
| `StartupModule()` | 模块加载到内存时 | 初始化资源、注册组件 |
| `ShutdownModule()` | 模块卸载时 | 清理资源 |

XGBlank 的 [模块声明](file:///d:/UPS/GitLab/XGUnrealNote/Courses/UE-CPP-Basic-to-Advanced/code/002_XGSampleTemp/Plugins/XGBlank/Source/XGBlank/Public/XGBlank.h)：

```cpp
class FXGBlankModule : public IModuleInterface
{
public:
    virtual void StartupModule() override;
    virtual void ShutdownModule() override;
};
```

## Content Only 纯内容插件

### 特点

- 只包含蓝图资产（`Content/` 目录下的 `.uasset`），没有 C++ 代码
- `.uplugin` 中**不包含 Modules 数组**
- 适合纯资源包（材质、模型、动画等）

### .uplugin 配置示例

[XGContentOnly.uplugin](file:///d:/UPS/GitLab/XGUnrealNote/Courses/UE-CPP-Basic-to-Advanced/code/002_XGSampleTemp/Plugins/XGContentOnly/XGContentOnly.uplugin)：

```json
{
    "FileVersion": 3,
    "CanContainContent": true,
    "Installed": false
}
```

注意：没有 `Modules` 字段，这是 Content Only 插件的核心特征。

### 蓝图资源

`Content/BP_XGContentOnlyActor.uasset` — 存放在插件 `Content/` 目录下的蓝图资产，可在项目中直接拖入关卡使用。

## Blueprint Function Library 蓝图函数库插件

### 特点

- 自动生成 `UBlueprintFunctionLibrary` 子类
- 静态函数通过 `UFUNCTION(BlueprintCallable)` 暴露给蓝图节点
- 函数参数中的 `UPARAM(ref)` 元数据可将参数标记为引用传递

### .uplugin 配置

[XGBlueprintLibrary.uplugin](file:///d:/UPS/GitLab/XGUnrealNote/Courses/UE-CPP-Basic-to-Advanced/code/002_XGSampleTemp/Plugins/XGBlueprintLibrary/XGBlueprintLibrary.uplugin)：

```json
{
    "Modules": [
        {
            "Name": "XGBlueprintLibrary",
            "Type": "Runtime",
            "LoadingPhase": "PreLoadingScreen"
        }
    ]
}
```

`LoadingPhase: "PreLoadingScreen"` 表示该模块在加载屏幕之前加载，适合需要尽早初始化的功能模块。

### 蓝图函数库模式

[头文件](file:///d:/UPS/GitLab/XGUnrealNote/Courses/UE-CPP-Basic-to-Advanced/code/002_XGSampleTemp/Plugins/XGBlueprintLibrary/Source/XGBlueprintLibrary/Public/XGBlueprintLibraryBPLibrary.h)：

```cpp
UCLASS()
class UXGBlueprintLibraryBPLibrary : public UBlueprintFunctionLibrary
{
    GENERATED_UCLASS_BODY()

    UFUNCTION(BlueprintCallable,
        meta = (DisplayName = "XGBlueprintLibraryTestFuntion",
                Keywords = "XGBlueprintLibrary sample test"),
        Category = "XGBlueprintLibraryTesting")
    static float XGBlueprintLibraryTestFuntion(float Param);
};
```

[实现](file:///d:/UPS/GitLab/XGUnrealNote/Courses/UE-CPP-Basic-to-Advanced/code/002_XGSampleTemp/Plugins/XGBlueprintLibrary/Source/XGBlueprintLibrary/Private/XGBlueprintLibraryBPLibrary.cpp)：

```cpp
float UXGBlueprintLibraryBPLibrary::XGBlueprintLibraryTestFuntion(float Param)
{
    return Param + 998.f;
}
```

### 蓝图函数库关键宏

| 宏/说明 | 用途 |
|---------|------|
| `UCLASS()` | 标记类为 UE 反射系统类 |
| `GENERATED_UCLASS_BODY()` | 生成构造函数声明（旧风格，与 `GENERATED_BODY()` 等价） |
| `UFUNCTION(BlueprintCallable)` | 在蓝图中生成可执行节点（有执行引脚） |
| `UFUNCTION(BlueprintPure)` | 在蓝图中生成纯函数节点（无执行引脚） |
| `Category = "名称"` | 蓝图面板中的分类目录 |
| `DisplayName = "名称"` | 蓝图节点的显示名称 |
| `Keywords = "关键词"` | 蓝图搜索关键词 |
