# 第二十九章：WebSocket 语音识别（STT）知识概览

## 章节说明

本章基于 WSS（WebSocket over TLS）协议，实现**实时语音转文字（STT/Speech to Text）** 功能。与第二十七章（HTTP 上传）和第三十章（TTS 语音合成）同属 XGSampleXFLink 插件，但采用**完全不同的通信模式**——WebSocket 长连接 + 二进制音频帧持续推送。

| 对比维度 | 第二十七章（HTTP） | 第二十九章（WebSocket STT） | 第三十章（WebSocket TTS） |
|---------|-----------------|-------------------------|-------------------------|
| 协议 | HTTP POST 请求-响应 | WebSocket 长连接持续通信 | WebSocket 长连接请求-响应 |
| 数据方向 | 上行：PNG 文件 | 上行：PCM 二进制音频帧 | 下行：Base64 音频数据 |
| 节点类型 | UBlueprintAsyncActionBase | UGameInstanceSubsystem + BPLibrary | UBlueprintAsyncActionBase |
| 鉴权方式 | HMAC-SHA256 URL 签名 | HMAC-SHA1 URL 参数（请求头） | HMAC-SHA256 URL 签名 |
| 流式特性 | 单次上传 | 实时帧推送（40ms/帧） | 单次发送，分段接收 |

## 整体 Pipeline

```
麦克风 → UAudioCapture（48K/44.1K）
   ↓ OnAudioGenerate（音频线程）
TArray<float> 缓冲区（FScopeLock 保护）
   ↓ GetVoiceData（消费线程轮询）
重采样：48K/44.1K → 16K mono
   ↓ SendVoiceData
float → int16 → uint8 LE bytes
   ↓ Socket->Send(binary)
WebSocket 长连接 → 讯飞 STT 服务器
   ↓ OnMessage
JSON 响应解析 → 蓝图委托回调
```

## 边界说明

- 本章处理的是**STT 语音识别**部分，同插件的 TTS 语音合并在第三十章
- 音频采集使用 UE 内置的 `UAudioCaptureFunctionLibrary`，而非平台特定 API
- WebSocket 使用 `FWebSocketsModule` 的 `IWebSocket` 接口
