# 摄像头视频流整合到 RenderTarget2D

本章使用 UE 内置的 **Media Framework** 打开摄像头设备，将视频帧渲染到 `URenderTarget2D`，作为 PNG 生成的输入源。

## 整体链路

```
UMediaSource (设备枚举)
    ↓ OpenUrl()
UMediaPlayer (播放控制)
    ↓ 输出到
UMediaTexture (GPU 纹理)
    ↓ 材质采样
UMaterial (MediaTexture 采样材质)
    ↓ DrawMaterialToRenderTarget
URenderTarget2D (CPU 可读的目标)
    ↓ ReadPixels
PNG Binary Data
```

## 核心步骤

### 1. 枚举摄像头设备

```cpp
TArray<FMediaSourceEntry> Sources;
UMediaBlueprintFunctionLibrary::EnumerateAudioCaptureDevices(Sources);
```

`UMediaBlueprintFunctionLibrary` 提供设备枚举函数，返回可用设备列表。

### 2. 打开设备 URL

```cpp
UMediaPlayer* MediaPlayer = CreateObject<UMediaPlayer>();
UMediaSource* MediaSource = CreateObject<UMediaSource>();
MediaPlayer->OpenUrl(DeviceURL);
```

- `UMediaPlayer::OpenUrl()` 接受设备 URL 字符串（格式为 `vidcap://...`）
- `UMediaSource` 封装设备源

### 3. 轨道选择

设备可能有多个轨道（音视频），需要手动选择视频轨道：

```cpp
// 监听 OnMediaOpened 事件
MediaPlayer->OnMediaOpened.AddDynamic(this, &OnMediaOpenedCallback);

void OnMediaOpenedCallback(FString OpenedUrl)
{
    // 选择视频轨道
    MediaPlayer->SelectTrack(EMediaPlayerTrack::Video, VideoTrackIndex);
}
```

### 4. 延迟播放

选择轨道后不能立即 Play()，需要等待轨道切换生效：

```cpp
// 延迟 0.2 秒后播放
FTimerHandle TimerHandle;
GetWorld()->GetTimerManager().SetTimer(TimerHandle, FTimerDelegate::CreateLambda([this]()
{
    MediaPlayer->Play();
}), 0.2f, false);
```

### 5. 渲染到 RenderTarget2D

通过材质将 `UMediaTexture` 采样后输出到 `URenderTarget2D`：

```cpp
// 创建材质实例
UMaterial* MediaMaterial = LoadObject<UMaterial>(nullptr, TEXT("/Game/Materials/MediaMaterial"));
UMaterialInstanceDynamic* MatInstance = UMaterialInstanceDynamic::Create(MediaMaterial, this);
MatInstance->SetTextureParameterValue(TEXT("MediaTexture"), MediaTexture);

// 绘制到 RenderTarget2D
DrawMaterialToRenderTarget(WorldContextObject, RenderTarget2D, MatInstance);
```

材质设置要点：
- 材质中放置 `MediaTexture` 采样节点
- 输出连接至 Emissive Color（用于 UI）或 Base Color（用于渲染目标）
- UI 显示：使用 `Image` 控件 → `SetBrushFromMaterial(MatInstance)`

## 性能注意事项

- UE 的 Media Framework 会占用 GameThread Tick，对性能有一定影响
- 对于高帧率或高性能场景，建议使用外部库（如 OpenCV）代替 Media Framework
- 视频轨道选择前不可调用 Play()，否则播放将失败

## 关键 API 参考

| API | 用途 |
|-----|------|
| `UMediaPlayer::OpenUrl()` | 打开摄像头设备 |
| `UMediaPlayer::SelectTrack()` | 选择视频轨道 |
| `UMediaPlayer::Play()` | 开始播放 |
| `UMediaBlueprintFunctionLibrary::EnumerateAudioCaptureDevices()` | 枚举设备 |
| `UMediaTexture` | 视频流纹理 |
| `DrawMaterialToRenderTarget()` | 材质 → RenderTarget2D |
