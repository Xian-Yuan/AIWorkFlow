# 第七章 基础案例

## 概述

第七章是项目实战章节，通过 5 个独立的案例项目（分布在 4 个代码目录）演示 UE C++ 的核心开发模式。本章重点是**数据驱动设计**、**组件架构**、**碰撞与移动系统**，以及**第一人称项目完整工作流**。

本章知识点覆盖：Actor 编辑快速入门 → 游戏摄像机切换 → 玩家控制的摄像机 → 组件碰撞与自定义移动 → 第一人称完整项目（发射物 + 角色 + HUD + 动画蓝图）。

## 核心设计模式汇总

| 模式 | 示例 | 说明 |
|------|------|------|
| 数据驱动设计 | FloatingActorModify | C++ 定义结构+逻辑，BeginPlay 应用 Blueprint 运行时赋值 |
| 输入转发 | XGCollidingPawn → MovementComponent | Pawn 只转发方向，不关心移动物理 |
| 逻辑组件 | UXGCollidingPawnMovementComponent | 无场景附着，只做逻辑计算，可复用 |
| Camera-to-World 变换 | Fire() 中 GunOffset 转换 | FTransform(Rotation).TransformPosition(Offset) |
| Struct+TArray 自动检测 | XGCameraDirectorModify | GetAllActorsOfClass + USTRUCT 封装摄像头信息 |
| Actor 生命周期 | XGFPSProjectile | InitialLifeSpan + 碰撞自动销毁 |
