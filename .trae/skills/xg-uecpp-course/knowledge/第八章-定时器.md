# 第八章：定时器

## 概述

本章讲解 Unreal Engine 定时器系统（`FTimerManager` / `FTimerHandle`）的核心概念与实战用法。定时器是 UE 中最常用的时间驱动机制之一，用于在指定时间间隔后执行回调函数，适用于冷却倒计时、周期性检测、延迟触发等场景。
