# 服务端业务逻辑实现

## 概述

服务端实现三个业务接口：版本检查、登录、注册。所有业务逻辑集中在 `UXGSampleServerHttpObject` 的三个 `Hanlde*` 方法中，通过 `DispatchHttpRequest` 根据 `EXGSampleRequestType` 枚举路由。

路径：[XGSampleServerHttpObject.cpp](../code/013_独立程序源码/XGSampleServer/Private/ServerObject/XGSampleServerHttpObject.cpp)

## 版本检查（CheckVersion）

```cpp
FXGSampleServerResponse UXGSampleServerHttpObject::HanldeCheckVersion(const FString& InJsonStr)
{
    FXGSampleServerResponse Response;
    FXGSampleServerCheckVersionResponseData ResponseData;
    ResponseData.Version = FXGSampleServerConfigManage::Get()->GetInfo().ServerVersion;

    FString ResponseJsonData;
    FJsonObjectConverter::UStructToJsonObjectString(ResponseData, ResponseJsonData);

    Response.Result = EXGSampleServerResult::Success;
    Response.Data = ResponseJsonData;
    return Response;
}
```

检查服务器版本，返回 INI 配置中的 `ServerVersion` 值。这是最轻量的接口，不需要请求数据，仅用于验证服务器是否正常运行。

## 登录（Login）

```cpp
FXGSampleServerResponse UXGSampleServerHttpObject::HanldeLogin(const FString& InJsonStr)
{
    FXGSampleServerResponse Response;
    FXGSampleServerLoginRequestData RequestData;

    // 1. 请求体反序列化
    if (!FJsonObjectConverter::JsonObjectStringToUStruct(InJsonStr, &RequestData, 0, 0))
    {
        Response.Result = EXGSampleServerResult::JsonAnalysisFail;
        return Response;
    }

    // 2. 参数非空校验
    if (RequestData.UserName.IsEmpty() || RequestData.Password.IsEmpty())
    {
        Response.Result = EXGSampleServerResult::ParamFail;
        return Response;
    }

    // 3. 验证用户名+密码
    FString OutUserID;
    if (!VerifyUserNameAndPassword(RequestData.UserName, RequestData.Password, OutUserID, Response))
    {
        return Response;
    }

    // 4. 登录成功，返回用户信息
    FXGSampleServerLoginResponseData ResponseData;
    ResponseData.UserID = OutUserID;
    ResponseData.UserName = RequestData.UserName;

    FString ResponseJsonData;
    FJsonObjectConverter::UStructToJsonObjectString(ResponseData, ResponseJsonData);

    Response.Result = EXGSampleServerResult::Success;
    Response.Message = TEXT("Login Success");
    Response.Data = ResponseJsonData;
    return Response;
}
```

### 登录验证核心

```cpp
bool UXGSampleServerHttpObject::VerifyUserNameAndPassword(
    const FString& InUserName, const FString& InPassword,
    FString& OutUserID, FXGSampleServerResponse& InResponse)
{
    for (auto& User : UserList)
    {
        if (User.UserName.Equals(InUserName))
        {
            FString EncryptedPassword = FMD5::HashAnsiString(*InPassword);
            if (User.Password.Equals(EncryptedPassword))
            {
                OutUserID = FString::Printf(TEXT("%i"), User.UserID);
                return true;
            }
            else
            {
                InResponse.Result = EXGSampleServerResult::PasswordFail;
                InResponse.Message = TEXT("Password Fail");
                return false;
            }
        }
    }
    InResponse.Result = EXGSampleServerResult::UserNotFount;
    InResponse.Message = TEXT("User Not Found");
    return false;
}
```

验证流程：
1. 遍历 `UserList`（`TArray<FXGSampleServerUserInfo>`）
2. 按用户名匹配用户
3. 对输入的密码做 MD5 哈希后与存储的哈希值比较
4. 用户名不存在返回 `UserNotFount`，密码错误返回 `PasswordFail`
5. 验证成功后返回用户 ID

## 注册（Register）

```cpp
FXGSampleServerResponse UXGSampleServerHttpObject::HanldeRegister(const FString& InJsonStr)
{
    FXGSampleServerResponse Response;
    FXGSampleServerRegisterRequestData RequestData;

    // 1. 请求体反序列化
    if (!FJsonObjectConverter::JsonObjectStringToUStruct(InJsonStr, &RequestData, 0, 0))
    {
        Response.Result = EXGSampleServerResult::JsonAnalysisFail;
        return Response;
    }

    // 2. 参数非空校验
    if (RequestData.UserName.IsEmpty() || RequestData.Password.IsEmpty()
        || RequestData.Phone.IsEmpty() || RequestData.Mail.IsEmpty())
    {
        Response.Result = EXGSampleServerResult::ParamFail;
        return Response;
    }

    // 3. 检查用户名是否重复
    for (auto& User : UserList)
    {
        if (User.UserName.Equals(RequestData.UserName))
        {
            Response.Result = EXGSampleServerResult::UserAlreadyExisting;
            Response.Message = TEXT("User Already Existing");
            return Response;
        }
    }

    // 4. 创建新用户
    FXGSampleServerUserInfo& UserInfo = UserList.AddDefaulted_GetRef();
    UserInfo.UserID = UserList.Num();  // 简单自增 ID
    UserInfo.UserName = RequestData.UserName;
    UserInfo.Password = FMD5::HashAnsiString(*RequestData.Password);
    UserInfo.Phone = RequestData.Phone;
    UserInfo.Mail = RequestData.Mail;
    UserInfo.RegisterTime = FDateTime::Now().ToString();  // 服务端时间

    // 5. 返回注册用户信息
    FXGSampleServerRegisterResponseData ResponseData;
    ResponseData.UserID = UserInfo.UserID;
    ResponseData.UserName = UserInfo.UserName;
    ResponseData.Phone = UserInfo.Phone;
    ResponseData.Mail = UserInfo.Mail;
    ResponseData.RegisterTime = UserInfo.RegisterTime;

    FString ResponseJsonData;
    FJsonObjectConverter::UStructToJsonObjectString(ResponseData, ResponseJsonData);

    Response.Result = EXGSampleServerResult::Success;
    Response.Message = TEXT("Register Success");
    Response.Data = ResponseJsonData;
    return Response;
}
```

注册流程：
1. 反序列化注册请求数据（用户名、密码、手机、邮箱）
2. 参数非空校验
3. 遍历 UserList 检查用户名是否已存在
4. 如果不存在，创建新用户并写入 UserList
5. 使用 `AddDefaulted_GetRef()` 避免临时变量拷贝
6. 通过 `FDateTime::Now()` 获取服务端时间作为注册时间
7. 返回新用户信息（含服务端分配的 UserID 和注册时间）

## 用户数据模型

```cpp
USTRUCT()
struct FXGSampleServerUserInfo
{
    GENERATED_BODY()
    int32 UserID;
    FString UserName;
    FString Password;       // MD5 哈希存储
    FString Phone;
    FString Mail;
    FString RegisterTime;   // 服务端时间
};
```

用户数据以 `TArray<FXGSampleServerUserInfo>` 形式存储在内存中，服务器重启后数据丢失（无持久化存储）。这是课程演示的简化设计，生产环境应使用数据库。

## 业务逻辑对比

| 维度 | 版本检查 | 登录 | 注册 |
|------|---------|------|------|
| 是否需要 Token | 需要 | 需要 | 需要 |
| 是否需要请求数据 | 不需要 | 用户名+密码 | 用户名+密码+手机+邮箱 |
| 校验逻辑 | 无 | 用户名+密码匹配 | 用户名唯一性检查 |
| 副作用 | 无 | 无 | 向 UserList 写入新用户 |
| 响应数据 | Version | UserID+UserName | UserID+UserName+Phone+Mail+RegisterTime |
