# 第十七章：多线程

## 章节概述

本章讲解 Unreal Engine 的多线程编程体系，涵盖从基础线程概念到高级任务依赖系统的完整知识。UE 的多线程方案包括 FRunnable（独立线程）、AsyncTask/Async（轻量异步）、FGraphEvent（任务依赖 DAG）和 ParallelFor（并行循环），开发者需要理解各方案的适用场景和共享线程池的阻塞风险。

## 关键知识点

1. UE 多线程的核心是 **TaskGraph 线程池**——所有系统共享，长时间任务会阻塞引擎
2. **FRunnable** 创建独立线程，完全独立于 TaskGraph，适合长期后台任务
3. **FCriticalSection + FScopeLock** 保护共享数据，**FThreadSafeBool**（或 std::atomic\<bool\>）保护简单标志
4. **AsyncTask** 轻量异步，**Async + TFuture** 支持获取异步结果
5. **FGraphEvent** 支持构建任务依赖 DAG，适合流水线作业
6. **ParallelFor** 并行循环，`ForceSingleThread` 标志用于性能对比测试
