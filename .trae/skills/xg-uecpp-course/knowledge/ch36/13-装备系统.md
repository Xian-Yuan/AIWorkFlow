# 装备系统

## 整体架构

装备系统的结构与背包系统对称，由数据组件 + 物品类 + UI 面板 + UI 插槽组成：

```
XGRPGEquipmentComponent（装备数据组件，Replicated）
    └── TArray<FXGRPGEquipmentItem> EquipmentItems（装备数组）
XGRPGEquipment（装备物品，继承自 UXGRPGItem）
    └── GameplayEffectClass（装备绑定的 GE）
UI_EquipmentPanel（装备面板）
    └── WrapBox（换行容器）
        └── UI_EquipmentSlot × N（装备插槽，支持拖拽交换）
```

## 装备数据（FXGRPGEquipmentItem）

```cpp
struct FXGRPGEquipmentItem
{
    UXGRPGEquipment* RPGEquimentItemPointer;  // 装备指针
    FActiveGameplayEffectHandle ActiveEquipmentEffectHandle;  // 装备激活的 GE Handle
    bool IsValid();
    void ResetSelf();
};
```

`ActiveEquipmentEffectHandle` 用于管理装备 GE 的生命周期——装备时应用 GE，卸下时移除。

## 装备组件（XGRPGEquipmentComponent）

### 核心接口

```cpp
class UXGRPGEquipmentComponent : public UXGRPGComponentBase
{
    void AddEquipmentItem(FXGRPGEquipmentItem NewItem);   // 添加装备
    void RemoveEquipmentItem(int32 EquipmentId);           // 移除装备
    void UndockEquipmentByID(int32 EquipmentId);           // 卸下装备
    void SwapEquipmentItem(int32 A, int32 B);              // 装备槽交换
    void SwapFromInvToEqu(int32 InvId, int32 EquId);       // 仓库→装备
    void SwapFromEquToInv(int32 EquId, int32 InvId);       // 装备→仓库
};
```

### 装备时 GE 应用

当装备一个物品时，调用应用它的 GE：

```cpp
void UXGRPGEquipmentComponent::AddEquipmentItem(FXGRPGEquipmentItem Item)
{
    if (Item.RPGEquimentItemPointer && ASC)
    {
        // 应用装备 GE
        FActiveGameplayEffectHandle Handle = ASC->ApplyGameplayEffectToSelf(
            Item.RPGEquimentItemPointer->GameplayEffectClass.GetDefaultObject(),
            1.0f,
            ASC->MakeEffectContext());
        
        Item.ActiveEquipmentEffectHandle = Handle;
        EquipmentItems.Add(Item);
    }
}
```

卸下装备时通过 `ActiveEquipmentEffectHandle` 移除对应的 GE。

## 装备物品（XGRPGEquipment）

```cpp
class UXGRPGEquipment : public UXGRPGItem
{
    UPROPERTY(EditDefaultsOnly, BlueprintReadOnly)
    TSubclassOf<UGameplayEffect> GameplayEffectClass;  // 绑定的 GE
};
```

每个装备物品在蓝图中配置一个 GE，装备时自动应用该 GE 修改角色的属性（如 +10 Attack、+5 Defense）。

## 装备 GE 的应用效果

装备 GE 通过 Modifier 直接修改 AttributeSet：

| GE Modifier | 效果 | 示例 |
|------------|------|------|
| Attack +5 | 增加攻击力 | 武器攻击力加成 |
| Defense +5 | 增加防御力 | 护甲防御力加成 |
| MaxHealth +50 | 增加最大生命值 | 装备提升生命上限 |

GE 使用 `Infinite` 持续时间，在卸下装备时通过 Handle 移除。

## UI 装备面板（UUI_EquipmentPanel）

```cpp
class UUI_EquipmentPanel : public UUI_Base
{
    UWrapBox* EquipmentArray;                       // 装备栏容器
```

`WrapBox` 自动换行，适合装备栏的网格布局。

### 跨系统交换

装备系统与背包系统之间的交换是核心交互：

- **SwapFromInvToEqu**：从背包拖到装备槽，从 `InventoryComponent` 移除物品，添加到 `EquipmentComponent`
- **SwapFromEquToInv**：从装备槽拖到背包，从 `EquipmentComponent` 卸下装备（移除 GE），添加回 `InventoryComponent`

## 注意事项

- 装备 GE 是 `Infinite` 类型，需要记录 `FActiveGameplayEffectHandle` 以便移除
- `UndockEquipmentByID` 内部通过 Handle 移除 GE 效果
- 装备面板的刷新逻辑与背包类似，重新生成所有 Slots
- 跨系统交换（仓库↔装备）需要同时操作两个组件，注意时序
- 装备 GE 的 Modifier 可以是 `Additive`（加法加成）或 `Multiply`（百分比加成）
