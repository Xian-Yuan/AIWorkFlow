# AI 模型配置

> 角色提示词生成的 AI 模型列表。用户可在对话中选择模型，或设为默认。

## 当前模型

| 模型ID | 显示名称 | 类型 | 默认 | 说明 |
|--------|---------|------|:---:|------|
| deepseek-v4 | DeepSeek V4 | 文本生成 | ✅ | 默认模型，稳定可靠 |
| deepseek-v4-flash | DeepSeek V4 Flash | 文本生成 | | 轻量快速，适合批量生成 |

## 添加新模型

在下方按格式追加：

```json
{
  "id": "模型ID",
  "name": "显示名称",
  "type": "文本生成/图像生成",
  "default": false,
  "description": "简要说明"
}
```

## 使用方法

对话中用户可以说：
- "用 DeepSeek V4 Flash 生成" → 临时切换
- "默认模型改为 Flash" → 永久切换
- "列出可用模型" → 查看列表
