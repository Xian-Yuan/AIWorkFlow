# 冲突解决规则表

> 确定性的冲突规则。**不依赖 AI 即兴判断。**
> 格式：`If A AND B → Result`

---

## 规则总表

### A. 渲染×画质 (R1-R2)

| # | 冲突 | 解决 |
|---|------|------|
| R1 | 像素风 + 4K | 画质降为 HD |
| R2 | 像素风 + HD | HD保留，细节降为2层 |

### B. 渲染×模版 (R3-R4)

| # | 冲突 | 解决 |
|---|------|------|
| R3 | 像素风 + 细节特写 | 细节特写降为"色块特写" |
| R4 | 厚涂 + 全设计稿 + 标准画质 | 画质升为 HD |

### C. 模版×角色设计 (R5-R6)

| # | 冲突 | 解决 |
|---|------|------|
| R5 | 武器拆分 + 角色无武器 | 替换为"空手姿势" |
| R6 | 全设计稿 + 像素风(低) | 降为角色+三视图 |

### D. 配色×模版 (R7)

| # | 冲突 | 解决 |
|---|------|------|
| R7 | 暗色系 + 全设计稿 | 背景改为深灰(#333) |

### E. 跨域通用原则 (R8-R10)

| # | 冲突 | 解决 |
|---|------|------|
| R8 | 任意MUST(A) ∧ MUST(B) 矛盾 | 优先级: 职业>体型>种族>配色>流派. 同级按步骤顺序 |
| R9 | 画质MUST vs 渲染MUST | 渲染MUST优先 |
| R10 | 模版MUST vs 画质MUST | 画质MUST优先 |

### F. 职业×武器 (R11-R14)

| # | 冲突 | 解决 |

| R11 | 刺客 + 大型武器(>角色身高50%) | 武器降为匕首/短剑/爪，保持刺客轻便设定 |

## 冲突检测顺序

```
1. 配色×配色 (R19)                    ← 先解决颜色冲突
2. 力量来源×力量来源 (R25)              ← 互斥力量来源
3. 职业×武器 (R11-R14)                 ← 职业和武器不匹配
4. 种族×体型 (R15-R16)                 ← 种族比例冲突
5. 战斗风格×武器 (R22)                  ← 风格和武器矛盾
6. 流派×职业 (R17-R18)                 ← 设计和职业冲突
7. 职业×服装 (R26-R29)                 ← 职业和服装不匹配
8. 流派×时代/材质 (R30)                 ← 流派的时代/材质冲突
9. 配色×职业 (R20-R21)                 ← 配色和职业冲突
10. 力量来源×配色 (R23-R24)              ← 力量颜色冲突
11. 渲染×画质 (R1-R2)
12. 渲染×模版 (R3-R4)
13. 模版×角色设计 (R5-R6)
14. 配色×模版 (R7)
15. 跨域通用 (R8-R10)                   ← 兜底
```

## 输出格式

```
[冲突] R{编号}: {冲突描述} → {解决方案}
```
