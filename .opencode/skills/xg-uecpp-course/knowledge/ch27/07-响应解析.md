# 响应解析

科大讯飞静默活体检测 API 的响应体解析分为两个层级：**一级服务（鉴权）** 和 **二级服务（业务逻辑）**。代码见 [SilentBiopsyAsyncAction.cpp OnHttpRespReceived()](file:///d:/UPS/GitLab/XGUnrealNote/Courses/UE-CPP-Basic-to-Advanced/code/001_XGSampleDemo/Plugins/XGSampleLink/Source/XGSampleXFLink/Private/AsyncAction/XGSampleXFSilentBiopsyAyncAction.cpp#L119-L233)。

## 解析流程

```
HTTP Response
    ↓ header.code == 200 ?
    ↓
Parse JSON → RawMessage
    ↓ header.code == 0 ?    ← 一级服务（鉴权）
    ↓ Yes
Decode payload.text (Base64 → UTF8)
    ↓
Parse JSON → TextMessage
    ↓ ret == 0 ?            ← 二级服务（业务逻辑）
    ├── Yes → OnSuccess(bPassed, Score, FaceRect)
    ├── ret != 0 → OnFail(ErrorCode)
    └── Parse Failed → OnFail
```

## 响应结构体定义

代码见 [XGSampleXFSilentBiopsyRespType.h](file:///d:/UPS/GitLab/XGUnrealNote/Courses/UE-CPP-Basic-to-Advanced/code/001_XGSampleDemo/Plugins/XGSampleLink/Source/XGSampleXFLink/Public/Type/SilentBiopsy/XGSampleXFSilentBiopsyRespType.h)。

### RawMessage — 原始响应

```cpp
struct FXGSampleXFSilentBiopsyRespMessage
{
    FXGSampleXFSilentBiopsyRespHeaderInfo header;  // code, message, sid
    FXGSampleXFBiopsyRespPayloadInfo payload;       // anti_spoof_result {text, ...}
};

struct FXGSampleXFSilentBiopsyRespHeaderInfo
{
    int32 code = -1;        // 0=鉴权成功，非零=鉴权失败
    FString message = "";   // 错误描述
    FString sid = "";       // 会话 ID
};
```

### TextMessage — 解码后的业务数据

```cpp
struct FXGSampleXFSilentBiopsyRespTextMessage
{
    int32 ret = -1;         // 0=成功，2005=未检测到人脸
    bool passed = false;    // 活体检测是否通过
    float score = -1.0;     // 置信度分数（0~1）
    int32 x = -1;           // 人脸矩形左上角 X
    int32 y = -1;           // 人脸矩形左上角 Y
    int32 w = -1;           // 人脸矩形宽度
    int32 h = -1;           // 人脸矩形高度
    FString thresholds = "";// 阈值信息
};
```

### RespInfo — 最终返回给蓝图的合并结构

```cpp
struct FXGSampleXFSilentBiopsyRespInfo
{
    bool bPassed = false;                   // 是否通过检测
    float Score = -1.0;                     // 置信度
    FXGSampleXFSilentBiopsyRespTextMessage TextMessage;  // 业务数据
    FXGSampleXFSilentBiopsyRespMessage RawMessage;       // 原始数据
};
```

`FXGSampleXFSilentBiopsyRespInfo` 同时保留 RawMessage 和 TextMessage，方便 BP 调试时检查原始响应。

## 解析代码流程

### Step 1 — 检查 HTTP 响应

```cpp
if (bSucceeded && HttpRequest->GetStatus() == EHttpRequestStatus::Succeeded && HttpResponse->GetResponseCode() == 200)
```

HTTP 层面必须同时满足：请求成功、状态为 Succeeded、响应码 200。

### Step 2 — 解析原始 JSON

```cpp
FXGSampleXFSilentBiopsyRespMessage RespMessage;
FJsonObjectConverter::JsonObjectStringToUStruct(ResponseJson, &RespMessage);
```

### Step 3 — 一级鉴权检查

```cpp
if (RespMessage.header.code == 0)
{
    // 鉴权通过 → 解码业务数据
}
else
{
    CallOnFail(AsyncID, false, RespMessage.header.message, RespInfo);
}
```

`header.code` 非零表示鉴权失败（API Key/Secret 错误，或 URL 签名不正确）。

### Step 4 — Base64 解码 payload text

```cpp
FString TextString = FXGSampleXFLinkBase::Base64UTF8StringToTCharString(
    RespMessage.payload.anti_spoof_result.text);
```

`Base64UTF8StringToTCharString()` 的实现（见 [XGSampleXFLinkBase.cpp](file:///d:/UPS/GitLab/XGUnrealNote/Courses/UE-CPP-Basic-to-Advanced/code/001_XGSampleDemo/Plugins/XGSampleLink/Source/ThirdParty/XGSampleXFLinkLibrary/Private/XGSampleXFLinkBase.cpp#L148-L164)）：

```cpp
FString FXGSampleXFLinkBase::Base64UTF8StringToTCharString(const FString& InBase64UTF8String)
{
    TArray<uint8> TextData;
    FBase64::Decode(InBase64UTF8String, TextData);
    return FUTF8ToTCHAR(reinterpret_cast<const UTF8CHAR*>(TextData.GetData()), TextData.Num());
}
```

### Step 5 — 解析业务 JSON

```cpp
FXGSampleXFSilentBiopsyRespTextMessage RespTextMessage;
bool bParseSuccess = FJsonObjectConverter::JsonObjectStringToUStruct(TextString, &RespTextMessage);
```

### Step 6 — 二级业务码检查

```cpp
if (RespTextMessage.ret == 0)
{
    // 检测成功：返回 bPassed, Score, FaceRect
    RespInfo.bPassed = RespTextMessage.passed;
    RespInfo.Score = RespTextMessage.score;
    CallOnSuccess(AsyncID, true, TEXT("SilentBiopsy has finished !"), RespInfo);
}
else
{
    // 检测失败：ret=2005 表示未检测到人脸
    CallOnFail(AsyncID, false, 
        TEXT("SilentBiopsy TextMessage has Error ! Error Code is :") + FString::FromInt(RespTextMessage.ret), 
        RespInfo);
}
```

## 常见错误码

| code (一级) | 含义 |
|-------------|------|
| 0 | 鉴权成功 |
| 非零 | API Key/Secret 错误，或签名不正确 |

| ret (二级) | 含义 |
|------------|------|
| 0 | 活体检测成功 |
| 2005 | 未检测到人脸 |
| 其他 | 业务层错误，需查询讯飞文档 |
