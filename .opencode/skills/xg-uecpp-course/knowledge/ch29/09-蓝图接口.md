# 蓝图接口

## BPLibrary 声明

`UXGSampleXFLinkBPLibrary` 继承自 `UBlueprintFunctionLibrary`，提供 5 个静态蓝图调用函数。所有函数都直接转发到 `UXGSampleSTTSubsystem`。

## 函数清单

| 函数 | 蓝图节点名称 | 说明 |
|------|-------------|------|
| XGBeginRealTimeSpeechToText | XGSampleBeginRealTimeSpeechToText | 开始实时语音识别 |
| XGStopRealTimeSpeechToText | XGSampleStopRealTimeSpeechToText | 正常停止识别 |
| XGForceToStopRealTimeSPeechToText | XGSampleForceToStopRealTimeSPeechToText | 强制停止识别 |
| XGIsRealTimeSpeechToTextWorking | XGSampleIsRealTimeSpeechToTextWorking | 检查是否正在识别 |
| XGXunFeiGetRealTimeSpeechToTextVolumeSize | XGSampleXunFeiGetRealTimeSpeechToTextVolumeSize | 获取当前麦克风音量 |

## 开始语音识别

```cpp
static void XGBeginRealTimeSpeechToText(
    UObject* WorldContextObject,
    FString InAppID,
    FString InAPIKey,
    FXGSampleSTTReqInfo InSTTReqInfo,
    FXGSampleInitSTTDelegate InSTTInitDelegate,    // 连接成功/失败 → 1 次
    FXGSampleSTTRespDelegate InSTTRespDelegate,     // 识别结果 → 多次
    FXGSampleSTTCloseDelegate InSTTCloseDelegate,   // 连接关闭 → 1 次
    FGuid& OutSTTAsyncID
);
```

参数说明：
| 参数 | 类型 | 说明 |
|------|------|------|
| InAppID | FString | 讯飞开放平台应用 ID |
| InAPIKey | FString | 讯飞 API Key |
| InSTTReqInfo | FXGSampleSTTReqInfo | 请求参数结构体 |
| InSTTInitDelegate | 委托 | 初始化回调（连接结果） |
| InSTTRespDelegate | 委托 | 识别结果回调（多次触发） |
| InSTTCloseDelegate | 委托 | 关闭回调 |
| OutSTTAsyncID | FGuid | 输出异步会话 ID |

```cpp
void UXGSampleXFLinkBPLibrary::XGBeginRealTimeSpeechToText(
    UObject* WorldContextObject,
    FString InAppID, FString InAPIKey,
    FXGSampleSTTReqInfo InSTTReqInfo,
    FXGSampleInitSTTDelegate InSTTInitDelegate,
    FXGSampleSTTRespDelegate InSTTRespDelegate,
    FXGSampleSTTCloseDelegate InSTTCloseDelegate,
    FGuid& OutSTTAsyncID)
{
    UGameInstance* GameInstance = WorldContextObject->GetGameInstance();
    UXGSampleSTTSubsystem* Subsystem = GameInstance->GetSubsystem<UXGSampleSTTSubsystem>();

    // 保存委托和参数
    Subsystem->OnSTTInitDelegate = InSTTInitDelegate;
    Subsystem->OnSTTRespDelegate = InSTTRespDelegate;
    Subsystem->OnSTTCloseDelegate = InSTTCloseDelegate;
    Subsystem->STTReqInfo = InSTTReqInfo;

    OutSTTAsyncID = FGuid::NewGuid();
    Subsystem->STTAsyncID = OutSTTAsyncID;

    Subsystem->BeginRealTimeSpeechToText(InAppID, InAPIKey);
}
```

实现动作：
1. 通过 `WorldContextObject` 获取 `UGameInstance`
2. 通过 `GetSubsystem<UXGSampleSTTSubsystem>()` 获取 STT 子系统
3. 保存三个委托参数到子系统
4. 生成唯一 `FGuid` 作为会话 ID
5. 调用子系统的 `BeginRealTimeSpeechToText`

## 停止语音识别

```cpp
static void XGStopRealTimeSpeechToText(const UObject* WorldContextObject);
```

- 发送结束帧（status=1）到服务器
- 等待服务器确认后自动关闭 WebSocket
- 触发 `OnClosed` → 回调 `CloseDelegate`

## 强制停止语音识别

```cpp
static void XGForceToStopRealTimeSPeechToText(const UObject* WorldContextObject);
```

- 直接关闭 WebSocket（不等待服务器确认）
- 立即停止音频采集和消费线程
- 清理所有资源

## 检查工作状态

```cpp
static bool XGIsRealTimeSpeechToTextWorking(const UObject* WorldContextObject);
```

- 返回 `STTStatus != Ready`（即正在连接或正在识别中）
- 可用于控制 UI 按钮的启用/禁用状态

## 获取音量

```cpp
static float XGXunFeiGetRealTimeSpeechToTextVolumeSize(UObject* WorldContextObject);
```

- 从 `UXGSampleAudioCaptureSubsystem` 获取当前音量
- 可在 Tick 中调用实现音量条动态显示

## 蓝图节点设计

所有函数都是纯静态函数，蓝图左侧**不需要**执行线（无 `FExec` pin），但需要 `WorldContextObject` 来获取 GameInstance：

```
[XGSampleBeginRealTimeSpeechToText]
    AppID ──────────╮
    APIKey ─────────┤
    STTReqInfo ─────┤
    InitDelegate ───┤
    RespDelegate ───┤
    CloseDelegate ──┤
    OutSTTAsyncID ──╯
```
