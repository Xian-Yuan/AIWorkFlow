# 音频采集子系统

## 概述

`UXGSampleAudioCaptureSubsystem` 负责从麦克风采集实时音频数据。使用 UE 内置的 `UAudioCaptureFunctionLibrary` 而非平台特定 API（如 WASAPI/CoreAudio），实现跨平台兼容。

## 单例模式

子系统通过静态指针暴露自身，供消费线程跨模块访问：

```cpp
static UXGSampleAudioCaptureSubsystem* XunFeiAudioCaptureSubsystemPtr;
```

通过 `GetXunFeiAudioCaptureSubsystem` 静态方法获取实例。

## 创建音频采集

```cpp
void UXGSampleAudioCaptureSubsystem::CreateAudioCapture()
{
    // UAudioCaptureFunctionLibrary 是 UE 内置音频捕获工厂
    AudioCapture = UAudioCaptureFunctionLibrary::CreateAudioCapture();
    if (AudioCapture)
    {
        AudioCapture->OnAudioCaptureToAudioSignal.AddDynamic(this, &UXGSampleAudioCaptureSubsystem::OnAudioGenerate);
        AudioCapture->StartCapturingAudio();
    }
}
```

- `UAudioCaptureFunctionLibrary::CreateAudioCapture()` 创建音频捕获实例
- `OnAudioGenerate` 是音频线程的回调，每收到一帧音频数据时触发
- `StartCapturingAudio()` 开始采集

## OnAudioGenerate 回调

```cpp
void UXGSampleAudioCaptureSubsystem::OnAudioGenerate(const TArray<float>& InAudio, int32 InNumSamples, int32 InNumChannels, int32 InSampleRate, double InStreamTime, double InAudioTime)
{
    // 1. 音量计算（RMS）
    float Volume = 0.0;
    for (int32 i = 0; i < InNumSamples; i++)
        Volume += InAudio[i] * InAudio[i];
    Volume = FMath::Sqrt(Volume / InNumSamples);
    AsyncTask(ENamedThreads::GameThread, [this, Volume]()
    {
        SetVolumeSize(Volume);
    });

    // 2. 追加音频数据到缓冲区（临界区保护）
    if (InNumSamples > 0 && InSampleRate > 0 && InNumChannels > 0)
    {
        FScopeLock Lock(&VoiceDataSection);
        // 追加到全局缓冲区
        AudioCaptureVoiceData.Append(InAudio.GetData(), InNumSamples);
        VoiceChannelNum = InNumChannels;
        VoiceSampleRate = InSampleRate;
    }
}
```

参数含义：
| 参数 | 说明 |
|------|------|
| InAudio | 浮点音频数据数组，值域 [-1.0, 1.0] |
| InNumSamples | 样本数（总样本数 = 帧数 × 通道数） |
| InNumChannels | 通道数（通常 1 mono 或 2 stereo） |
| InSampleRate | 采样率（通常 48000 或 44100） |

## 数据缓冲区

- `AudioCaptureVoiceData`：`TArray<float>` 类型，累积所有未消费的音频数据
- `VoiceDataSection`：`FCriticalSection` 保护，防止音频线程和消费线程同时访问
- `AppendVoiceData` 在音频线程（OnAudioGenerate）写入
- `GetVoiceData` 在消费线程读取

## 停止采集

```cpp
void UXGSampleAudioCaptureSubsystem::StopCapturingAudio()
{
    if (AudioCapture)
    {
        AudioCapture->StopCapturingAudio();
        AudioCapture->OnAudioCaptureToAudioSignal.RemoveAll(this);
        AudioCapture = nullptr;
    }
    ClearVoiceData();
}
```

## 完整接口

| 方法 | 说明 |
|------|------|
| `CreateAudioCapture()` | 创建并启动音频采集 |
| `StopCapturingAudio()` | 停止采集并清理 |
| `AppendVoiceData(TArray<float>&)` | 追加音频数据（带临界区保护） |
| `GetVoiceData(TArray<float>&) → bool` | 取出音频数据并重采样（见重采样文档） |
| `ClearVoiceData()` | 清空缓冲区 + 重置音量 |
| `SetVolumeSize(float)` | 设置当前音量（游戏线程） |
| `GetVolumeSize() → float` | 获取当前音量 |
