# 商品详情页面配置

## 概述

商品详情页面是用户在虚幻商城上看到的插件展示页面，包含主图、缩略图、功能图、标题、定价、分类、标签、描述等信息。所有内容必须严格遵循商城规则。

---

## 图片素材规范

| 图片类型 | 尺寸 | 说明 |
|---------|------|------|
| **主图（Main Image）** | 1920×1080 | 商品详情页顶部大图 |
| **缩略图（Thumbnail）** | 由主图自动裁剪 | 商城列表页展示 |
| **功能图（Feature Graphics）** | 1920×600（约） | 用于特色展示区域 |

### 图片内容规则

- 必须使用 **UE 引擎截图**，不能使用外部渲染软件（如 Maya、Blender）渲染的图片
- 不能包含非 UE 的 UI 元素，如 Windows 任务栏、桌面图标、浏览器窗口等
- 首次提交时**所有图片必须为英文**（不能包含中文文字）
- 图片内容需真实反映插件功能，不得过度美化或虚假宣传
- 参考商城已上架的高质量商品图风格

---

## 商品标题规则

### 禁止使用的词汇

商品标题**不得包含**以下商标名或受保护名称：

- `UE`、`Unreal Engine`、`Unreal`
- `Apex`、`Apex Launcher`
- 国家名称（`China`、`American` 等）
- 其他已知商标名

### 标题命名规范

- 标题应准确描述插件功能
- 避免使用夸大宣传用语
- 推荐格式：`[功能描述] - [插件类型]`（如 `Advanced Inventory System - Code Plugin`）

## 定价策略

| 项目 | 说明 |
|------|------|
| **货币单位** | USD（美元） |
| **Epic 抽成** | 12%（Epic 抽取销售额的 12%） |
| **免费策略** | 可以免费发布，但仅限于插件类型 |
| **定价参考** | 参考商城同类插件的定价区间 |

---

## 分类与标签

### 分类（Category）

| 类别 | 说明 |
|------|------|
| **Code Plugin** | 包含 C++ 源码的插件 |
| **Non-Code Plugin** | 纯蓝图、数据、资源类插件（不包含 C++ 代码） |

- 分类决定了用户在商城筛选时的可见范围
- 包含任何 C++ 代码的插件必须选 Code Plugin

### 标签（Tags）

- 选择与插件功能相关的标签（如 `Inventory`、`Animation`、`AI` 等）
- 标签帮助用户在搜索时找到你的插件
- 不要使用不相关的标签

### 商品描述

- 首次提交时**描述必须为英文**
- 描述内容需清晰说明插件的功能和使用场景
- 可以对功能特点进行结构化列举
- 不得包含链接或其他联系方式
- 不得包含虚假或误导性信息

---

## 第三方库归属声明

如果插件使用了第三方库，必须进行声明：

1. 在商品详情中**如实标注**使用的所有第三方软件
2. 在审核流程中填写 **Third-Party Software Attribution Form**
3. 表单包含：
   - 第三方库名称
   - 开源协议类型
   - 官方项目 URL
   - 作者联系方式
4. **Openssl 被明确标记为问题库**，使用它可能导致审核被拒

> 首次提交者最容易忽视的两点：第三方库归属声明和文件版权声明。
