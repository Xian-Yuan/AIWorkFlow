# 插件类型概述与工程配置

## 概述

本章覆盖了 Unreal Engine 插件开发的 **7 种类型**，依托 `XGSampleTemp` 项目下的 7 个插件示例进行讲解。课程核心主题是**虚幻标准下的第三方库封装**。

---

## 7 种插件类型

| 类型 | .uplugin Type 字段 | 说明 | 代码位置 |
|------|-------------------|------|---------|
| **Blank（空插件）** | Runtime | 最简插件骨架，仅包含 StartModule/ShutdownModule | [XGBlank](file:///d:/UPS/GitLab/XGUnrealNote/Courses/UE-CPP-Basic-to-Advanced/code/002_XGSampleTemp/Plugins/XGBlank/) |
| **Content Only（纯内容插件）** | 无 Modules | 只包含蓝图资源，没有 C++ 代码 | [XGContentOnly](file:///d:/UPS/GitLab/XGUnrealNote/Courses/UE-CPP-Basic-to-Advanced/code/002_XGSampleTemp/Plugins/XGContentOnly/) |
| **Blueprint Function Library** | Runtime | 暴露 C++ 静态函数给蓝图调用 | [XGBlueprintLibrary](file:///d:/UPS/GitLab/XGUnrealNote/Courses/UE-CPP-Basic-to-Advanced/code/002_XGSampleTemp/Plugins/XGBlueprintLibrary/) |
| **Editor Mode（编辑器模式）** | Editor | 自定义编辑器交互模式 | [XGEditorMode](file:///d:/UPS/GitLab/XGUnrealNote/Courses/UE-CPP-Basic-to-Advanced/code/002_XGSampleTemp/Plugins/XGEditorMode/) |
| **Editor Standalone Window** | Editor | 在编辑器菜单中打开独立 Slate 窗口 | [XGEditorStandaloneWindow](file:///d:/UPS/GitLab/XGUnrealNote/Courses/UE-CPP-Basic-to-Advanced/code/002_XGSampleTemp/Plugins/XGEditorStandaloneWindow/) |
| **Editor Toolbar Button** | Editor | 在编辑器工具栏添加自定义按钮 | [XGEditorToolbarButton](file:///d:/UPS/GitLab/XGUnrealNote/Courses/UE-CPP-Basic-to-Advanced/code/002_XGSampleTemp/Plugins/XGEditorToolbarButton/) |
| **Third Party Library（第三方库）** | Runtime | 在插件中集成外部原生库（DLL/.lib） | [XGThirdPartyLibrary](file:///d:/UPS/GitLab/XGUnrealNote/Courses/UE-CPP-Basic-to-Advanced/code/002_XGSampleTemp/Plugins/XGThirdPartyLibrary/) |

### 编辑器插件说明

- `Type: "Editor"` 的插件在**打包时会被自动剥离**，仅用于编辑器开发环境
- 编辑器插件依赖 Slate UI 框架，开发门槛高于 Gameplay 但低于引擎底层开发
- 本章对编辑器类型插件仅做创建和展示，不深入展开编辑器开发

## 工程配置

### 编辑器偏好设置

创建插件前，需要开启编辑器插件扩展点显示：

- 路径：**Editor Preferences → General → Miscellaneous → Display UI Extension Points**
- 勾选后才能在菜单中看到插件创建的 UI 扩展点

## 创建插件的通用流程

1. 右键 Content Browser → **Plugins** → 点击 **Add New**
2. 选择对应插件模板
3. 命名插件（使用 XG 前缀以标识为课程插件）
4. 确认 `.uplugin` 文件生成到 `项目根目录/Plugins/插件名/`
5. 在 `.uplugin` 中补全 `FriendlyName`、`Category`、`Description` 等字段
6. 编译并确认插件加载成功
