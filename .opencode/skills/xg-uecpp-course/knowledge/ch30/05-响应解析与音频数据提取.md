# TTS 响应解析与音频数据提取

## 响应数据结构

```cpp
USTRUCT(BlueprintType)
struct FXGXunFeiTTSRespDataInfo
{
    GENERATED_BODY()
    UPROPERTY(BlueprintReadWrite)
    FString audio;    // Base64 编码的音频数据

    UPROPERTY(BlueprintReadWrite)
    int32 status;     // 数据状态：2=完成

    UPROPERTY(BlueprintReadWrite)
    FString ced;      // 编码描述
};

USTRUCT(BlueprintType)
struct FXGXunFeiTTSRespInfo
{
    GENERATED_BODY()
    UPROPERTY(BlueprintReadWrite)
    int32 code;       // 状态码：0=成功

    UPROPERTY(BlueprintReadWrite)
    FString message;  // 描述信息

    UPROPERTY(BlueprintReadWrite)
    FString sid;      // 会话 ID

    UPROPERTY(BlueprintReadWrite)
    FXGXunFeiTTSRespDataInfo data;
};
```

### JSON 示例

正常响应：
```json
{
    "code": 0,
    "message": "success",
    "sid": "tts00000001",
    "data": {
        "audio": "//uQxAAA...<base64 encoded audio data>...",
        "status": 2,
        "ced": "UTF-8"
    }
}
```

## 解析逻辑

```cpp
void UXGSampleTTSAsyncAction::OnWebSocketMessage(const FString& InStr)
{
    FXGXunFeiTTSRespInfo RespInfo;
    if (!FJsonObjectConverter::JsonObjectStringToUStruct(InStr, &RespInfo))
    {
        OnSoundWaveFail.Broadcast(TEXT("响应 JSON 解析失败"));
        RealeaseResources();
        return;
    }

    if (RespInfo.code != 0)
    {
        // 服务器返回错误
        FString ErrorMsg = FString::Printf(TEXT("服务器错误: %d - %s"), RespInfo.code, RespInfo.message);
        OnSoundWaveFail.Broadcast(ErrorMsg);
        RealeaseResources();
        return;
    }

    // Base64 解码音频数据
    TArray<uint8> AudioChunk;
    if (!FBase64::Decode(RespInfo.data.audio, AudioChunk))
    {
        OnSoundWaveFail.Broadcast(TEXT("音频 Base64 解码失败"));
        RealeaseResources();
        return;
    }

    // 累积到总音频数据
    AllAudioData.Append(AudioChunk);

    // 检查是否为最后一帧
    if (RespInfo.data.status == 2)
    {
        ProcessCompleteAudio();
    }
}
```

## 音频数据累积

`AllAudioData` 是一个 `TArray<uint8>` 成员变量，用于累积所有音频数据块：

```cpp
TArray<uint8> AllAudioData;
```

由于 TTS 服务器通常在单次响应中返回所有数据（status=2），`AllAudioData` 主要用于：
1. 支持分段传输场景（理论上可能存在多帧）
2. 在 `ProcessCompleteAudio` 中将完整音频数据传递给后续处理
