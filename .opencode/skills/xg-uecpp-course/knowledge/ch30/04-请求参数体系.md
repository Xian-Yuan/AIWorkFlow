# TTS 请求参数体系

## 三层嵌套结构

讯飞 TTS API 的请求参数分为三层：

```json
{
    "common": { ... },      // 公共参数（应用 ID）
    "business": { ... },    // 业务参数（音色/语速/音量等）
    "data": { ... }          // 数据参数（Base64 文本）
}
```

## 数据结构定义

```cpp
// 公共参数
USTRUCT(BlueprintType)
struct FXGXunFeiTTSReqCommonInfo
{
    GENERATED_BODY()
    UPROPERTY(BlueprintReadWrite)
    FString app_id;  // 应用 ID（protected，只读）
};

// 业务参数
USTRUCT(BlueprintType)
struct FXGXunFeiTTSReqBusinessInfo
{
    GENERATED_BODY()
    UPROPERTY(BlueprintReadWrite)
    FString aue = "raw";        // 音频编码：raw/lame

    UPROPERTY(BlueprintReadWrite)
    FString auf = "audio/L16;rate=16000";  // 音频格式

    UPROPERTY(BlueprintReadWrite)
    FString vcn = "xiaoyan";    // 发音人（音色）

    UPROPERTY(BlueprintReadWrite)
    int32 speed = 50;           // 语速 [0~100]，默认 50

    UPROPERTY(BlueprintReadWrite)
    int32 volume = 50;          // 音量 [0~100]，默认 50

    UPROPERTY(BlueprintReadWrite)
    int32 pitch = 50;           // 音调 [0~100]，默认 50

    UPROPERTY(BlueprintReadWrite)
    int32 bgs = 0;              // 背景音 [0~100]，默认 0

    UPROPERTY(BlueprintReadWrite)
    FString tte = "UTF8";       // 文本编码格式：UTF8/GB2312

    UPROPERTY(BlueprintReadWrite)
    FString reg = "2";          // 文本格式化：2（数字读法调整）

    UPROPERTY(BlueprintReadWrite)
    FString rdn = "0";          // 合成风格：0（普通）
};

// 数据参数
USTRUCT(BlueprintType)
struct FXGXunFeiTTSReqDataInfo
{
    GENERATED_BODY()
    UPROPERTY(BlueprintReadWrite)
    FString text;               // Base64 编码后的文本

    UPROPERTY(BlueprintReadWrite)
    int32 status = 2;           // 固定为 2（一次性发送所有文本）
};

// 完整请求
USTRUCT(BlueprintType)
struct FXGXunFeiTTSReqInfo
{
    GENERATED_BODY()
    UPROPERTY(BlueprintReadWrite)
    FXGXunFeiTTSReqCommonInfo common;

    UPROPERTY(BlueprintReadWrite)
    FXGXunFeiTTSReqBusinessInfo business;

    UPROPERTY(BlueprintReadWrite)
    FXGXunFeiTTSReqDataInfo data;
};
```

## 业务参数详解

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| aue | string | "raw" | 音频编码格式。`raw` = 原始 PCM，`lame` = MP3 编码 |
| auf | string | "audio/L16;rate=16000" | 音频格式：L16（16bit 线性 PCM），采样率 16000 |
| vcn | string | "xiaoyan" | 发音人。可选：xiaoyan（小燕）、aisjping、aisbabyxu 等 |
| speed | int | 50 | 语速，范围 [0, 100] |
| volume | int | 50 | 音量，范围 [0, 100] |
| pitch | int | 50 | 音调，范围 [0, 100] |
| bgs | int | 0 | 背景音编号，0 表示无背景音 |
| tte | string | "UTF8" | 文本编码类型。UTF8 / GB2312 |
| reg | string | "2" | 数值格式化：2（自动读法调整）|
| rdn | string | "0" | 合成风格：0（普通合成）|

## 请求发送流程

```cpp
// 1. 填充请求结构体
FXGXunFeiTTSReqInfo ReqInfo;
ReqInfo.common.app_id = InAppID;
ReqInfo.business.vcn = "xiaoyan";
// ...

// 2. Base64 编码文本
FString UTF8Str = TCHAR_TO_UTF8(InText);
ReqInfo.data.text = FBase64::Encode(UTF8Str);
ReqInfo.data.status = 2; // 一次发送所有文本

// 3. UStruct → JSON 字符串
FString JsonStr;
FJsonObjectConverter::UStructToJsonObjectString(ReqInfo, JsonStr);

// 4. 通过 WebSocket 发送
Socket->Send(JsonStr);
```

## 文本编码转换

`FString` 内部为 UTF-16 编码，需转为 UTF-8 再进行 Base64：

```cpp
FString UTF8Str = TCHAR_TO_UTF8(InText); // FString → UTF-8
ReqInfo.data.text = FBase64::Encode(UTF8Str);
```
