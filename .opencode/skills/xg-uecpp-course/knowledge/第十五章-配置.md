## 知识点速览

1. **DeveloperSettings** — 继承 UDeveloperSettings，在 Project Settings UI 中可视化编辑配置
2. **三个覆盖方法** — GetContainerName（Project/Editor/Engine）、GetCategoryName（侧边栏分类）、GetSectionName（节名）
3. **Config 标记** — UCLASS(Config = X) 决定 INI 段名，UPROPERTY(Config) 标记参与序列化的属性
4. **C++ 读写** — GetMutableDefault 获取 CDO，Modify() 标记脏，SaveConfig 写入指定 INI
5. **INI 6 层层级** — Engine Base → Engine Platform → Project Default → Project Platform → Saved/User → Packaged，高层覆盖低层
6. **GConfig 手动读写** — GetInt/SetInt + Flush，支持 GGameIni 句柄或直接文件路径
7. **密钥泄露** — INI 在 pak 中明文存储，启用 pak 加密或使用服务端密钥管理
8. **扩展内容** — Console Variables、命令行参数、Exec 命令、Cheat Manager
