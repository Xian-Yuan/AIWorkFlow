# FP（第一人称）模板详解

## 概述

本章基于虚幻引擎官方 First Person 模板，手写实现了完整的 FP 第一人称射击体系，涵盖蓝图解读与 C++ 编码两个阶段。核心架构：**GameMode → PlayerController → Character → PickupComponent → WeaponComponent → Projectile**。

## 增强输入系统（Enhanced Input）

### 输入资产

| Input Action | 类型 | 用途 |
|---|---|---|
| `IA_Jump` | Boolean | 跳跃 |
| `IA_Move` | 2D Axis | WASD 移动 |
| `IA_Look` | 2D Axis | 鼠标视角 |
| `IA_Shoot` | Boolean | 开火 |

| Input Mapping Context | 用途 |
|---|---|
| `IMC_Default` | 跳跃 + 移动 + 视角（基础常驻） |
| `IMC_Weapons` | 开火（装备武器后注册） |

### IMC 按键映射

- **IMC_Default**：Jump → 空格 (Triggered)；Move → W/A/S/D 各带对应 Modifier（Negate/SwapAxis）；Look → 鼠标 XY 2D 轴 (Y 轴 Negate)
- **IMC_Weapons**：Shoot → 鼠标左键 (Pressed)

### 注册流程（PlayerController::BeginPlay）

```cpp
void AFPC_SimplePlayerController::BeginPlay()
{
    Super::BeginPlay();
    if (UEnhancedInputLocalPlayerSubsystem* Subsystem = ULocalPlayer::GetSubsystem<UEnhancedInputLocalPlayerSubsystem>(GetLocalPlayer()))
    {
        Subsystem->AddMappingContext(IMC_Default, 0);
    }
}
```

### C++ 输入绑定（Character::SetupPlayerInputComponent）

```cpp
void AFPC_SimpleCharacter::SetupPlayerInputComponent(UInputComponent* PlayerInputComponent)
{
    UEnhancedInputComponent* EnhancedInputComponent = Cast<UEnhancedInputComponent>(PlayerInputComponent);
    if (!EnhancedInputComponent) { UE_LOG(LogTemp, Error, TEXT("...")) return; }

    EnhancedInputComponent->BindAction(JumpAction, ETriggerEvent::Started, this, &ACharacter::Jump);
    EnhancedInputComponent->BindAction(JumpAction, ETriggerEvent::Completed, this, &ACharacter::StopJumping);
    EnhancedInputComponent->BindAction(MoveAction, ETriggerEvent::Triggered, this, &AFPC_SimpleCharacter::Move);
    EnhancedInputComponent->BindAction(LookAction, ETriggerEvent::Triggered, this, &AFPC_SimpleCharacter::Look);
}
```

### 移动逻辑

```cpp
void AFPC_SimpleCharacter::Move(const FInputActionInstance& Instance)
{
    FVector2D Value = Instance.GetValue().Get<FVector2D>();
    FRotationMatrix RotMatrix(Controller->GetControlRotation());
    AddMovementInput(RotMatrix.GetUnitAxis(EAxis::Y), Value.X);   // 右
    AddMovementInput(RotMatrix.GetUnitAxis(EAxis::X), Value.Y);   // 前
}
```

### 视角逻辑

```cpp
void AFPC_SimpleCharacter::Look(const FInputActionInstance& Instance)
{
    FVector2D Value = Instance.GetValue().Get<FVector2D>();
    AddControllerYawInput(Value.X);
    AddControllerPitchInput(Value.Y);
}
```

## Character 构造函数

```cpp
AFPC_SimpleCharacter::AFPC_SimpleCharacter()
{
    GetCapsuleComponent()->InitCapsuleSize(34.f, 88.f);   // FP 精度：缩小胶囊体

    SpringArmComp = CreateDefaultSubobject<USpringArmComponent>(TEXT("SpringArm"));
    SpringArmComp->SetupAttachment(GetCapsuleComponent());
    SpringArmComp->bUsePawnControlRotation = true;

    CameraComp = CreateDefaultSubobject<UCameraComponent>(TEXT("Camera"));
    CameraComp->SetupAttachment(SpringArmComp);

    Mesh1P = CreateDefaultSubobject<USkeletalMeshComponent>(TEXT("Mesh1P"));
    Mesh1P->SetupAttachment(CameraComp);
    Mesh1P->SetOnlyOwnerSee(true);
    Mesh1P->bCastDynamicShadow = false;
    Mesh1P->CastShadow = false;
}
```

## Pickup 拾取机制

- **PickupComponent**（继承 USphereComponent）：BeginPlay 绑定 `OnComponentBeginOverlap`，重叠时 Cast 到 Character，触发动态多播委托 `OnPickUp`
- **拾取逻辑**：检测是否已持有武器 → 未持有则调用 WeaponComponent::AttachWeapon → 销毁 Pickup 自身

## WeaponComponent 武器组件

继承 USkeletalMeshComponent，核心变量：

| 变量 | 用途 |
|---|---|
| `TSubclassOf<AActor> ProjectileClass` | 投掷物类 |
| `USoundBase* FireSound` | 开火音效 |
| `UAnimMontage* FireMontage` | 开火蒙太奇 |
| `FVector MuzzleOffset` | 枪口偏移 |
| `UInputAction* FireAction` | 开火 Input Action |

### AttachWeapon

```cpp
void UFPC_SimpleWeaponComponent::AttachWeapon(ACharacter* TargetCharacter)
{
    if (AlreadyHasWeapon()) return;  // 防止重复装备
    SetOwnerCharacter(TargetCharacter);
    AttachToComponent(TargetCharacter->GetMesh1P(), FAttachmentTransformRules::SnapToTargetIncludingScale, TEXT("GripPoint"));
    // 注册武器 IMC
    if (auto* Subsystem = ULocalPlayer::GetSubsystem<UEnhancedInputLocalPlayerSubsystem>(TargetCharacter->GetController<APlayerController>()->GetLocalPlayer()))
    {
        Subsystem->AddMappingContext(WeaponMappingContext, 1);
    }
}
```

### Fire（开火）

```cpp
void UFPC_SimpleWeaponComponent::Fire()
{
    // 获取枪口位置（从 Socket）
    FVector MuzzleLocation = GetSocketLocation(TEXT("Muzzle"));
    // 生成投掷物，设置 Instigator
    AActor* Projectile = GetWorld()->SpawnActor<AActor>(ProjectileClass, MuzzleLocation, GetOwner()->GetActorRotation());
    Projectile->SetInstigator(GetOwner<APawn>());
    // 播放音效与蒙太奇
    PlaySound(FireSound);
    PlayMontage(FireMontage);
}
```

### 资源清理

WeaponComponent 被销毁时，应移除注册的 IMC（通过 `OnDestroy` 或 Owner 的 EndPlay 处理）。

## Projectile 投掷物

继承 AActor，核心组件：
- **SphereComponent**：RootComponent，物理碰撞，碰撞预设 BlockAll
- **ProjectileMovementComponent**：弹道模拟，设置反弹/速度/初速参数
- **InitialLifeSpan = 3**：3 秒自动销毁

```cpp
void AFPC_SimpleProjectile::OnHit(UPrimitiveComponent* HitComp, AActor* OtherActor,
    UPrimitiveComponent* OtherComp, FVector NormalImpulse, const FHitResult& Hit)
{
    if (OtherActor && OtherComp && OtherComp->IsSimulatingPhysics())
    {
        OtherComp->AddImpulse(GetVelocity() * 100.f);
    }
    Destroy();
}
```

## 碰撞设置要点

| 组件 | 碰撞预设 | 原因 |
|---|---|---|
| WeaponComponent Mesh | NoCollision | 避免遮挡射线检测 |
| Projectile SphereComponent | BlockAll | 确保能碰撞到场景物体 |
| Character Capsule | 默认 Pawn | 基础碰撞 |

## 蓝图配置

所有 C++ 类的具体资产引用（Mesh/Sound/Montage 等）在蓝图中配置，C++ 只声明变量引用和写入核心逻辑。需要创建的蓝图子类：BP_FPC_SimpleCharacter、BP_FPC_SimplePlayerController、BP_FPC_SimpleGameMode、BP_FPC_SimpleProjectile。
