## 一、课程全景

| 维度 | 数据 |
|------|------|
| 总章节数 | **37 章**（含第零章宣传视频） |
| 总课时数 | **400+ 课时** |
| 总时长 | **约 100 小时** |
| 总代码量 | **4 万+ 行**（核心插件 ~14,000 行 + 项目源码 ~11,000 行 + 其他项目 ~33,000 行） |
| 究极案例 | **12 个** |
| 编程语言 | 绝大多数 C++，极少蓝图 |
| 前置要求 | 扎实蓝图基础 + 一定 C++ 基础（面向对象、指针、引用、STL） |
| 适用人群 | 0-1 年 UE 经验强烈推荐；1-3 年推荐查漏补缺；零基础不推荐 |
| 课程目标 | 达到高级/资深虚幻客户端开发者门槛 |
| 引擎版本 | UE 5.4（发布版 + 源码版双模式） |
