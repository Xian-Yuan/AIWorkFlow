# 压缩符号速查

## 约束标记
◆ = MUST (必须) ｜ ✕ = FORBID (禁止) ｜ ◇ = SHOULD (建议)
H = 硬约束 ｜ S = 软约束 ｜ ↯X = 与X冲突

## 数值
≥ = 至少/大于等于 ｜ ≤ = 最多/小于等于 ｜ %N = 占比N% ｜ → = 导致/结果是

## 视觉
👁 = 视觉符号 ｜ 🎨 = 配色约束 ｜ 8bit: = 低像素适配 ｜ ◆✕ = 不适合

## 示例对比

原文 (68 tokens):
```
视觉符号: 垂直悬垂布料，全身覆盖，飘逸感
MUST: 服装为连体长袍(从头到脚或至少到小腿); 布料有自然垂坠褶皱; 下摆宽松
FORBID: 紧身设计; 金属护甲大块覆盖
配色约束: 无强制
低像素适配: 纵向大色块+底部喇叭形展开(3-4格宽)
```

压缩 (35 tokens, 压缩比 49%):
```
👁连体长袍·垂坠·宽松
◆连体≥小腿 ◆自然褶皱 ◆下摆宽松
✕紧身/金属甲>10%
8bit:纵长色块+底展3-4格
```

## 压缩比目标
- 单文件: 40-60% 压缩
- 全约束集: 50-65% 压缩
- 目标: 约束0丢失，语义100%保留
