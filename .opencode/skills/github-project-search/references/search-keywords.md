## 搜索语法速查

| 语法 | 示例 | 效果 |
|------|------|------|
| 引号精确匹配 | `"computer use"` | 不拆分单词 |
| 语言过滤 | `language:python` | 只搜 Python |
| 组合 | `MCP stars:>100` | Stars > 100 的 MCP 项目 |
| 排除 | `-deprecated` | 排除废弃项目 |
