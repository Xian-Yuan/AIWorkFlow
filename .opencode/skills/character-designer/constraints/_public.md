# 公共约束（始终生效）

> 这些约束在任何设计中都生效，不依赖用户的具体选择。

## 核心原则

- **剪影优先**: 远看必须能识别职业与角色类型
- **功能与美学平衡**: 装饰不能阻碍动作；防护与灵活平衡
- **元素统一**: 主元素占 >50% 视觉比重，副元素 <30%，禁止均分
- **轮廓可读**: 每增加一个细节，必须能回答"它为什么在这里"

## 通用设计规则

- 主色 >50% / 辅色 20-30% / 点缀色 <10%
- 角色重心在画面中上 1/3
- 武器和手的关系必须可读
- 面部是视觉焦点，头部附近的装饰 ≤2 个强对比元素

## 风格兼容

- 像素风：所有约束自动加上"16x16可识别"检查
- 写实风：所有约束自动加上"材质光照合理性"检查
